## 📖 How These Courses Are Designed

| Pillar | What It Looks Like | Why It Matters |

| 🍿 **Stories & Analogies** | Pizza shops, road trips, Netflix shows, sports teams | Your brain remembers stories 22× better than facts |

## 🎬 How Each Module Works

Every module folder contains four files:

```
Module-XX-Topic/
├── Reading.md       ← The main lesson (story-driven, ~25 min)
├── Videos.md        ← Curated YouTube search links (~30–60 min)
├── Quiz.md          ← ≥24 questions with explanations (engine auto-shuffles)
└── Cheat-Sheet.md   ← One- or two-page summary for review
```

Each course folder ALSO has:

- `README.md` — overview, exam facts, recommended path
- `Flashcards.md` — interactive widget with 60–150 Q/A cards
- `Practice-Exams/` — 2 partial practice exams + 1 full-length mock matching the real cert format

**Recommended flow per module:**
1. ☕ Read the `Reading.md` (with coffee)
2. 🎥 Watch 1–2 videos from `Videos.md`
3. ✏️ Take the `Quiz.md` *without peeking*
4. 📋 Review the `Cheat-Sheet.md`
5. 🃏 Add new terms to your flashcards (Anki recommended — see [Resources](./Resources/index.md))
6. 😴 Sleep — that's literally when memory consolidates

## 🆘 When You Get Stuck

- **Concept won't stick?** → Try teaching it out loud to your dog/cat/plant. The "Feynman Technique" is undefeated.
- **Practice question wrong?** → Don't just check the answer. Ask: *"What would have to be true for the wrong answer to be right?"*
- **Burned out?** → Take a day off. Two days off. Your brain is *consolidating*, not slacking.
