## ⏱️ Suggested Order

**If you have 4 months:** Do Scrum first (it's a foundation for the agile portion of PMP), then PMP.
**If you have 6 weeks:** Pick ONE. Don't try both. Seriously.
**If you have unlimited time:** Do them in parallel — but be warned, the terminology overlaps and can confuse you.

## 🎬 How To Use Each Module

Every module folder contains four files:

```
Module-XX-Topic/
├── Reading.md       ← The main lesson (read this first, ~20-30 min)
├── Videos.md        ← Curated YouTube links (watch in order, ~30-60 min)
├── Quiz.md          ← 10-15 questions with explanations
└── Cheat-Sheet.md   ← One-page summary for review
```

**Recommended flow per module:**
1. ☕ Read the `Reading.md` (with coffee — pretend you're enjoying yourself)
2. 🎥 Watch 1–2 videos from `Videos.md`
3. ✏️ Take the `Quiz.md` *without peeking*
4. 📋 Print/review the `Cheat-Sheet.md`
5. 🃏 Add new terms to your flashcards (Anki recommended — see Resources)
6. 😴 Sleep — that's literally when memory consolidates

## 🆘 When You Get Stuck

- **Concept won't stick?** → Try teaching it out loud to your dog/cat/plant. The "Feynman Technique" is undefeated.
- **Practice question wrong?** → Don't just check the answer. Ask: *"What would have to be true for the wrong answer to be right?"*
- **Burned out?** → Take a day off. Two days off. Your brain is *consolidating*, not slacking.
