# Module 5 Cheat Sheet — Surah Al-Fatiha

## The Surah at a Glance

| Feature | Detail |
|---------|--------|
| Number | Surah 1 |
| Verses | 7 |
| Classification | Makki (revealed in Mecca) |
| Other names | Umm al-Quran, Umm al-Kitab, Al-Sab' al-Mathani |
| Daily recitations | Min. 17 times (in 5 prayers) |

---

## Full Text

| # | Arabic | Transliteration | English |
|---|--------|-----------------|---------|
| 1 | بِسۡمِ ٱللَّهِ ٱلرَّحۡمَٰنِ ٱلرَّحِيمِ | Bismi-llāhi r-raḥmāni r-raḥīm | In the name of Allah, the Merciful, the Compassionate |
| 2 | ٱلۡحَمۡدُ لِلَّهِ رَبِّ ٱلۡعَٰلَمِينَ | Al-ḥamdu li-llāhi rabbi l-ʿālamīn | Praise be to Allah, Lord of all the worlds |
| 3 | ٱلرَّحۡمَٰنِ ٱلرَّحِيمِ | Ar-raḥmāni r-raḥīm | The Entirely Merciful, the Especially Merciful |
| 4 | مَٰلِكِ يَوۡمِ ٱلدِّينِ | Māliki yawmi d-dīn | Master of the Day of Recompense |
| 5 | إِيَّاكَ نَعۡبُدُ وَإِيَّاكَ نَسۡتَعِينُ | Iyyāka naʿbudu wa iyyāka nastaʿīn | You alone we worship; You alone we ask for help |
| 6 | ٱهۡدِنَا ٱلصِّرَٰطَ ٱلۡمُسۡتَقِيمَ | Ihdinā ṣ-ṣirāṭa l-mustaqīm | Guide us to the straight path |
| 7 | صِرَٰطَ ٱلَّذِينَ أَنۡعَمۡتَ عَلَيۡهِمۡ... | Ṣirāṭa lladhīna anʿamta ʿalayhim... | The path of those You have blessed... |

## Tajweed Points

| Verse | Word | Rule |
|-------|------|------|
| 1 | اللَّهِ after بِسمِ | Thin Lam (Tarqeeq) — Kasra before it |
| 1 | الرَّحمَٰن | Thick Ra (Fatha); long Alif (Madd Tabee'i) |
| 4 | الدِّينِ | Shamsiyyah Lam (absorbed into Dal) |
| 4 | مَٰلِك | Long Alif (Madd Tabee'i) |
| 6 | الصِّرَٰطَ | Shamsiyyah Lam (absorbed into Sod) |
| 7 | الضَّآلِّين | Madd Lazim (6 beats before Shaddah) |
| 7 | عَلَيهِم | Madd Leen (Ya + Sukoon after Fatha) |

## Key Vocabulary

| Arabic | Meaning |
|--------|---------|
| الحَمد | Praising intrinsic qualities (deeper than thanks) |
| رَبّ | Creator + Owner + Nurturer |
| العَٰلَمِين | All worlds / all planes of existence |
| الرَّحمَٰن | Mercy for all creation |
| الرَّحِيم | Mercy specific to believers |
| مَٰلِك / مَلِك | Master-Owner / King (two valid readings) |
| الدِّين | Recompense / Day of Judgment |
| الصِّرَاط | Path / Way |
| الهِدَايَة | Guidance |

## Tafseer Quick Notes

| Scholar | Key Insight |
|---------|------------|
| Al-Mizan (Tabatabai) | All real praise flows back to Allah — hamd for creation is hamd for Creator |
| Namoona (Shirazi) | إِيَّاكَ نَعبُدُ = declaration of total spiritual freedom |
| Tasnim (Jawadi Amoli) | "Guide us" = request for continuous, deepest divine guidance |
