## 🌐 Platform Map (Owner + Algorithmic Lever)

| Platform | Owner | What ranks |
|---|---|---|
| **Instagram** | Meta | Reels + interest/recency/relationship/frequency |
| **Facebook** | Meta | Friends/groups + Reels |
| **Threads** | Meta | Following + recommendations |
| **TikTok** | ByteDance | Completion + rewatch + hook + niche affiliation (NOT followers) |
| **YouTube** | Alphabet | CTR + AVD + retention + session time |
| **LinkedIn** | Microsoft | Dwell time + comments + recency (suppresses outbound links) |
| **X** | X Corp | Engagement velocity + Premium boost |
| **Pinterest** | Pinterest | Freshness + visual relevance (DRIVES OUTBOUND) |
| **Snapchat** | Snap Inc. | Friends + Discover |
| **Reddit** | Reddit | Subreddit-specific upvotes |

## 🎯 Top Metrics

| Metric | Formula / Definition |
|---|---|
| **Engagement rate** | (likes + comments + shares) ÷ followers (or impressions) |
| **Reach** | Unique users who saw it |
| **Impressions** | Total display count |
| **CTR** | Clicks ÷ impressions |
| **Completion rate** | % who watched to end |
| **Dwell time** | Seconds paused on post |

## 🚨 Common Traps

- ❌ Believing follower count drives TikTok ranking (it doesn't)
- ❌ Posting blog links directly in LinkedIn (downranked, put link in first comment)
- ❌ Treating Pinterest like Instagram (different, drives outbound)
- ❌ Spreading thin across 7 platforms
- ❌ Buying followers (engagement falls; algorithm punishes)
- ❌ Skipping FTC disclosure on AI-generated endorsements
