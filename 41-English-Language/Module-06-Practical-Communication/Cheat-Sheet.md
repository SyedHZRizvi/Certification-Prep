# Module 6 Cheat Sheet — Practical Communication (B1-B2)

---

## Phone Call Phrases

| Situation | Phrase |
|-----------|--------|
| Opening (caller) | "Hello, could I speak to [name], please?" |
| Opening (receiver) | "Good morning, [Company]. How can I help you?" |
| Person unavailable | "I'm afraid he's in a meeting / not available right now." |
| Leave a message | "Could I leave a message?" / "Would you like to leave a message?" |
| Ask to call back | "Could you ask her to call me back at her earliest convenience?" |
| Speak slowly | "Would you mind speaking a little more slowly?" |
| Didn't hear | "I'm sorry, could you repeat that?" / "Could you say that again, please?" |
| Ending | "Thank you for your time. Goodbye." |

## Discourse Markers — Complete Table

| Function | Discourse Marker | Example |
|----------|-----------------|---------|
| Adding a point | Furthermore / Moreover / In addition | **Furthermore**, the data confirms our hypothesis. |
| Contrasting | However / Nevertheless / On the other hand | Sales rose; **however**, profits fell. |
| Conceding | Admittedly / Despite this | **Admittedly**, there are challenges. |
| Cause/Reason | Because / Due to / Owing to | Late **due to** heavy traffic. |
| Result/Effect | Therefore / Consequently / As a result | Traffic was bad; **consequently**, I was late. |
| Exemplifying | For instance / For example / Such as | **For instance**, carbon emissions rose 5%. |
| Summarising | In summary / In conclusion / To sum up | **In conclusion**, the project was a success. |
| Sequencing | First / Then / Subsequently / Finally | **First**, analyse the data. **Then**, report. |
| Emphasising | Indeed / In fact / Clearly | **In fact**, the results were better than expected. |

## Expressing Opinions — Ready Phrases

| Introducing opinion | "In my view..." / "From my perspective..." / "I would argue that..." |
|---|---|
| Agreeing | "I entirely agree with..." / "That's a valid point." |
| Partial agreement | "You have a point, however..." / "I see where you're coming from, but..." |
| Disagreeing politely | "I take your point, but I'm not entirely convinced." / "With respect, I think..." |
| Emphasising | "What I find particularly important is..." |
| Concluding | "All things considered..." / "On balance, I believe..." |

---

## Modal Verbs — Practical Communication Uses

| Modal | Meaning | Formal Example |
|-------|---------|---------------|
| could | Polite request | "Could you send me the report?" |
| would | Polite offer/request | "Would you like to reschedule?" |
| should | Recommendation | "You should confirm in writing." |
| must | Strong obligation | "All invoices **must** be signed." |
| may | Formal permission/possibility | "You **may** submit the form online." |
| might | Suggestion / possibility | "We **might** consider a third option." |
| will | Commitment / offer | "I **will** send the proposal by Friday." |

## Presentation Language

| Stage | Phrases |
|-------|---------|
| Opening | "Today, I'd like to talk to you about..." / "The aim of my presentation is to..." |
| Signposting | "Moving on to..." / "Turning now to..." / "This brings me to..." |
| Referring to visuals | "As you can see from this chart..." / "This graph illustrates..." |
| Inviting questions | "I'd be happy to take questions at the end." |
| Summarising | "To summarise the key points..." / "In conclusion..." |
| Closing | "Thank you for your attention. Are there any questions?" |
