## 🎯 Answers + Explanations

| # | Answer | Explanation |
|---|--------|-------------|
| 1 | **B** | "She is **a** doctor" — indefinite article before singular noun. Urdu has no articles so this is often skipped! |
| 2 | **B** | "Elephant" starts with vowel sound /ɛ/ → use "an". |
| 3 | **B** | "The sun" — we use "the" when there is only one of something. |
| 4 | **B** | "Umbrella" starts with vowel sound /ʌ/ → "an umbrella". |
| 5 | **C** | child → children (irregular plural). Never "childrens"! |
| 6 | **B** | city → cities (consonant before y → change y to ies). |
| 7 | **D** | foot → feet (irregular plural). |
| 8 | **C** | Object pronoun after a verb = him (not he). "I gave **him** the book." |
| 9 | **B** | Possessive pronoun (standing alone) = "hers". "her" needs a noun after it (her bag). |
| 10 | **C** | "He plays football every Sunday" — habit, uses Present Simple. |
| 11 | **C** | study → studies (she/he/it: y → ies). |
| 12 | **B** | "right now" = Present Continuous: am/is/are + verb-ing. |
| 13 | **B** | "They are playing" — correct form. A: "is sleep" should be "is sleeping". |
| 14 | **C** | "in the box" — inside a container/space → "in". |
| 15 | **B** | "on the table" — on a surface → "on". |
| 16 | **C** | "at home" — fixed expression. Always "at home", not "in home". |
| 17 | **C** | "Who" asks about a person. "What" asks about a thing. |
| 18 | **D** | "Where" asks about place/location. |
| 19 | **B** | "Why" asks for reason/explanation. |
| 20 | **B** | "How many" for countable nouns (students can be counted). "How much" = uncountable. |
| 21 | **D** | English = Subject-Verb-Object. Urdu = Subject-Object-Verb. |
| 22 | **C** | "She eats food every day" — Subject (She) + Verb (eats) + Object (food). |
| 23 | **C** | In negative with does not, the verb returns to base form: "doesn't **sleep**". |
| 24 | **C** | In questions with does, verb is base form: "Does he **drink**?" |
| 25 | **B** | A/an = indefinite article (غیر معروف). The = definite article (معروف). |
| 26 | **D** | "under" is a preposition (shows relationship/position). Run=verb, happy=adjective, quickly=adverb. |
