# Module 4 Cheat Sheet — Everyday Conversations (A2/B1)

> Conversation phrase book organized by situation. Memorize the bold phrases first.

---

## Shopping

| Goal | Phrase |
|------|--------|
| Find an item | **Excuse me, where can I find ___?** |
| Ask price | **How much does this cost?** / How much is this? |
| Ask for variation | Do you have this in a different size/color? |
| Pay by card | **Can I pay by card?** |
| Return item | **I'd like to return this, please.** |
| Ask for receipt | Can I get a receipt, please? |
| Keep change | Keep the change. |

---

## Restaurant / Café

| Goal | Phrase |
|------|--------|
| Get a table | **A table for two, please.** |
| See menu | Can I see the menu, please? |
| Order food | **I'd like the [dish], please.** |
| Order drinks | **Can I have a glass of water?** |
| Ask sides | What comes with this? |
| Get recommendation | What do you recommend? |
| Get bill | **Could we have the bill, please?** |
| Problem with order | Excuse me, this is not what I ordered. |

---

## Directions

| Goal | Phrase |
|------|--------|
| Ask directions | **Excuse me, how do I get to ___?** |
| Say you're lost | I'm lost. Can you help me? |
| Ask distance | Is it far from here? |

| Direction | Phrase |
|-----------|--------|
| سیدھا آگے | Go straight ahead. |
| دائیں مڑیں | Turn right. |
| بائیں مڑیں | Turn left. |
| سڑک پار کریں | Cross the road. |
| کونے پر | It's on the corner. |
| کے پاس | It's next to ___. |

## Public Transport

| Goal | Phrase |
|------|--------|
| One-way ticket | **A single ticket to ___, please.** |
| Return ticket | **A return ticket to ___, please.** |
| Ask about bus | Which bus goes to ___? |
| Ask price | How much is the fare to ___? |
| Ask stop | Where do I get off for ___? |
| Ask for alert | Could you tell me when we reach ___? |

## At the Doctor

| Symptom | Phrase |
|---------|--------|
| سر درد | I have a headache. |
| پیٹ درد | I have a stomachache. |
| بخار | I have a fever. |
| کھانسی | I have a cough. |
| گلا خراب | I have a sore throat. |
| چکر | I feel dizzy. |
| متلی | I feel nauseous. |
| سینے میں درد | I have chest pain. |
| درد یہاں | It hurts here. |
| تین دنوں سے | I've had this for three days. |
| الرجی | I'm allergic to ___. |

## Phone Conversations

| Goal | Phrase |
|------|--------|
| Answer | Hello? |
| Identify yourself | **This is [name] speaking.** |
| Ask for someone | **Can I speak to ___?** |
| Ask them to wait | Hold on, please. / One moment. |
| Person not in | I'm sorry, he/she is not available. |
| Take a message | Can I take a message? |
| Leave a message | **Can I leave a message?** |
| Ask to slow down | Could you speak more slowly, please? |
| Ask to repeat | Could you repeat that, please? |

---

## Invitations

| Accepting | Declining |
|-----------|----------|
| That sounds great! | I'm afraid I can't make it. |
| I'd love to! | I'm sorry, I have other plans. |
| Count me in! | Maybe next time? |
| Sure, I'll be there. | I wish I could, but I'm busy. |

---

## Opinions and Feelings

| Giving Opinion | Agreement / Disagreement |
|---------------|-------------------------|
| I think that... | I agree with you. |
| In my opinion, ... | That's a good point. |
| I believe... | I see what you mean, but... |
| I feel that... | I disagree. |
| I'm not sure, but... | You make a good argument. |
