## 🎯 Answers + Explanations

| # | Answer | Explanation |
|---|--------|-------------|
| 1 | **B** | "How much does this cost?" is the standard way to ask price. |
| 2 | **C** | "I'd like to return this, please" — polite request to return an item. |
| 3 | **B** | "I'd like the chicken, please" — polite ordering phrase. Just saying "Give me" sounds demanding. |
| 4 | **C** | "Could we have the bill, please?" — polite way to ask for the check. |
| 5 | **B** | "Excuse me, I'm lost. Can you help me?" — appropriate when you need directions. |
| 6 | **B** | Turn right = دائیں مڑیں. Turn left = بائیں مڑیں. |
| 7 | **B** | "A single ticket" = one-way. "A return ticket" = round trip. |
| 8 | **C** | "I have a fever" — correct structure: I have + symptom. |
| 9 | **A** | "It hurts here" while pointing is the natural way to describe pain location. |
| 10 | **B** | "This is Sara speaking" — standard phone self-identification. |
| 11 | **C** | "Can I leave a message?" — you want to give information to pass on. |
| 12 | **C** | "Could you speak more slowly, please?" — polite request. |
| 13 | **C** | "I'd love to! That sounds great!" — enthusiastic acceptance. |
| 14 | **C** | "I'm afraid I can't make it. I have other plans." — most polite decline with explanation. |
| 15 | **B** | "In my opinion, ..." is the standard opener for sharing a personal view. |
| 16 | **A** | "That's a good point. I agree with you." — acknowledges their point and agrees. |
| 17 | **B** | "Fire! Everybody get out!" — clear emergency alert. |
| 18 | **A** | "I've been robbed! I need the police!" — correct emergency response. |
| 19 | **C** | "Which bus goes to the city center?" — asking about the right bus route. |
| 20 | **B** | "Could you tell me when we reach Central Station?" — asking to be alerted at a stop. |
| 21 | **B** | Asking about a different size = looking for a variation of the same item. |
| 22 | **B** | "Keep the change" = telling the cashier they can keep the remainder after payment. |
| 23 | **C** | "I wish I could, but I'm busy" = regretful, polite decline. |
| 24 | **B** | "I see what you mean, but..." = partial disagreement while acknowledging the other view. |
| 25 | **C** | "A return ticket" = round trip (go and come back). |
| 26 | **C** | "I've had this for three days" = the problem started 3 days ago and is still happening. |
