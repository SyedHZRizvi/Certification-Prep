## Nominalization — Verb to Noun Pairs

| Verb | Nominalization | Example (formal) |
|------|---------------|-----------------|
| decide | decision | "The **decision** was made unanimously." |
| analyse | analysis | "The **analysis** revealed key trends." |
| develop | development | "The **development** of the product took two years." |
| implement | implementation | "Full **implementation** is expected by Q3." |
| recommend | recommendation | "Our **recommendation** is to restructure." |
| fail | failure | "The **failure** of the system caused delays." |
| invest | investment | "A significant **investment** was required." |
| improve | improvement | "Measurable **improvements** were observed." |
| produce | production | "**Production** will increase by 20%." |
| achieve | achievement | "This **achievement** was recognised globally." |
| require | requirement | "The key **requirements** are listed below." |
| propose | proposal | "The **proposal** was accepted by the board." |

## Hedging Language — Full Toolkit

| Strength | Hedging Phrase | Use |
|----------|---------------|-----|
| Strong hedge | "The evidence suggests that..." | Academic reports |
| Medium hedge | "It would appear that..." | Business analysis |
| Medium hedge | "This tends to indicate..." | Research conclusions |
| Medium hedge | "It is widely believed that..." | General statements |
| Weak hedge | "It might be argued that..." | Tentative claims |
| Weak hedge | "There is some evidence to suggest..." | Uncertain findings |
| Approximating | "approximately / roughly / around" | Figures |
| Limiting scope | "In most cases..." / "Under certain conditions..." | Generalizations |
| Attribution | "According to [source]..." | Citing data |

## Meeting Language — Functional Phrases

| Function | Phrase |
|----------|--------|
| Opening | "I'd like to call the meeting to order." / "Let's get started." |
| Agenda | "The first item on the agenda is..." |
| Inviting input | "What are your thoughts on this?" / "Does anyone have anything to add?" |
| Interrupting politely | "Sorry to cut in, but..." / "If I may, I'd like to add..." |
| Clarifying | "Could you elaborate on that point?" / "What exactly do you mean by...?" |
| Summarising | "So, to recap what we've agreed..." |
| Action points | "Ahmed will send the report by Monday." |
| Postponing | "Let's table this and revisit it next week." |
| Closing | "I think we've covered everything. Thank you all." |

## Formal Writing Register — Transformations

| Informal | Formal |
|----------|--------|
| I'm sorry we can't | We regret that we are unable to |
| We need to talk about | It is necessary to address |
| Thanks for your email | Thank you for your correspondence |
| Get in touch | Please do not hesitate to contact us |
| I want to ask about | I am writing to enquire about |
| Start | Commence / Initiate |
| Use | Utilise / Employ |
| End | Conclude / Terminate |
| Show | Demonstrate / Illustrate |
| Help | Assist / Support |
