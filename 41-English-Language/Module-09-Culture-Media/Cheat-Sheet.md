## Figurative Language — Types Table

| Device | Definition | Example |
|--------|-----------|---------|
| Simile | Comparison using "like" or "as" | "She ran **like the wind**." |
| Metaphor | Direct comparison without "like/as" | "Life is a **journey**." |
| Personification | Giving human qualities to non-human things | "The wind **whispered** through the trees." |
| Hyperbole | Deliberate exaggeration | "I've told you a **million times**." |
| Alliteration | Repetition of initial consonant sounds | "**P**eter **P**iper **p**icked a **p**eck..." |
| Oxymoron | Two contradictory terms together | "**bittersweet**" / "**deafening silence**" |
| Irony (situational) | Outcome is opposite of what was expected | "The fire station burned down." |
| Euphemism | Mild expression for something unpleasant | "passed away" (= died) |
| Symbolism | Using an object to represent an idea | A dove = peace |
| Onomatopoeia | Words that sound like what they describe | "buzz", "crash", "hiss" |

## Literary Terms — Quick Reference

| Term | Definition | Example |
|------|-----------|---------|
| Theme | Central idea of a work | Love, justice, power |
| Tone | Author's attitude toward the subject | Ironic, melancholic, hopeful |
| Narrator | The voice telling the story | First-person ("I"), third-person ("he/she") |
| Plot | The sequence of events | Beginning → Rising action → Climax → Resolution |
| Setting | Time and place of the story | Victorian London |
| Protagonist | Main character | Harry Potter |
| Antagonist | Character opposing the protagonist | Voldemort |
| Climax | The turning point of maximum tension | The final confrontation |
| Foreshadowing | Hints at future events | Dark clouds before a storm scene |
| Flashback | A scene set earlier than the main narrative | A character remembering childhood |
