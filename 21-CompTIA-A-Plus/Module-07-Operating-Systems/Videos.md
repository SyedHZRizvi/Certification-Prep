## ✅ After watching

1. List the 4 main Windows desktop editions and one feature each.
2. Recite the Windows 11 hardware requirements (TPM, UEFI, Secure Boot, CPU).
3. Explain when to use FAT32 vs exFAT vs NTFS.
4. Name 5 Linux commands and their Windows equivalents.
5. Walk through booting into Windows Recovery Environment and running `bootrec`.
