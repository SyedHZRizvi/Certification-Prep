---

## 🔄 Change Types

| Type | Approval | Speed |
|------|----------|-------|
| **Standard** | Pre-approved template | Fast |
| **Normal** | CAB review | Days–weeks |
| **Emergency** | Expedited + post-CAB | Same-day |

✅ Description + Reason
✅ Impact analysis
✅ Risk level
✅ Test plan + results
✅ **Backout plan**
✅ Maintenance window
✅ Approvals (CAB members)
✅ Implementation steps
✅ Post-change verification
✅ Documentation updates (KB, CMDB, runbook)

---

## ⚡ ESD Protection

- Anti-static wrist strap (grounded to chassis or ESD mat)
- ESD mat
- Anti-static bag for boards
- Self-ground by touching unpainted metal
- Low humidity = more static

🎯 First action with any board: wrist strap on.

---

## 🔌 UPS Types

| Type | Description |
|------|-------------|
| Standby | Cheap; brief switch gap |
| Line-interactive | Mid; auto-voltage adjust |
| Online | Premium; zero switch gap |

🚨 Surge protector ≠ UPS. UPS = batteries + surge.

---

## 🌡️ Server Room Targets

- Temp: **18–27 °C / 64–80 °F** (ASHRAE)
- Humidity: **8–60% RH**
- Cable mgmt: Velcro ties (not zip)
- Fire: clean-agent (FM-200, Novec 1230)

---

## ☣️ Disposal

| Item | Disposal |
|------|----------|
| Li-ion battery | Electronics recycler |
| Lead-acid UPS | Hazardous waste / supplier |
| CR2032 (CMOS) | E-waste |
| Toner cartridge | Vendor program |
| Toner spill | HEPA vacuum + mask |
| MSDS / SDS | Look up at manufacturer |

## 🏆 Exam Power Phrases

✅ Often **right**:
- "Document the findings (Step 6)"
- "Follow company policy"
- "Get CAB approval before high-risk change"
- "Backout plan before implementation"
- "Anti-static wrist strap first"
- "Recycle batteries — don't trash"
- "Verify with the user before closing the ticket"

❌ Often **wrong**:
- "Skip documentation"
- "Argue with the customer"
- "Browse user files for fun"
- "Install pirated software for the user"
- "Skip CAB for production change"
- "Vacuum toner with a regular vacuum"

## 🗓️ Memorize Cold

- 6-step methodology Step 6 = Document
- ESD wrist strap = first hardware-handling action

- 18–27 °C server room target
- MSDS / SDS for chemicals
- PII / PHI / PCI / CUI
