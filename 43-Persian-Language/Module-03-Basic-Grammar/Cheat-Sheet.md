# Module 3 Cheat Sheet — بنیادی گرامر (Basic Grammar)

---

## Verb "To Be" (بودن — *budan*)

### Full Forms (هستن)
| Person | Persian | Transliteration | English |
|---|---|---|---|
| I | هستم | *hastam* | I am |
| You (inf.) | هستی | *hasti* | You are |
| He/She/It | است | *ast* | He/she/it is |
| We | هستیم | *hastim* | We are |
| You (form./pl.) | هستید | *hastid* | You are |
| They | هستند | *hastand* | They are |

### Short/Clitic Forms
| Suffix | Example | Translation |
|---|---|---|
| ـم (-am) | خوبم | I am good |
| ـی (-i) | خوبی | You are good |
| است (ast) | خوب است | He/she is good |
| ـیم (-im) | خوبیم | We are good |
| ـید (-id) | خوبید | You (pl.) are good |
| ـند (-and) | خوبند | They are good |

> Colloquial: است → ـه (-e): خوبه = "it's good"

## Present Tense Conjugation

**Formula: می‌ + [present stem] + [personal ending]**

### Personal Endings
| Person | Ending | Example (رفتن) | English |
|---|---|---|---|
| I | ـم (-am) | می‌روم | I go |
| You (sg.) | ـی (-i) | می‌روی | You go |
| He/She/It | ـد (-ad) | می‌رود | He/she goes |
| We | ـیم (-im) | می‌رویم | We go |
| You (pl./form.) | ـید (-id) | می‌روید | You go |
| They | ـند (-and) | می‌روند | They go |

### Key Verb Stems
| Infinitive | Present Stem | Past Stem | English |
|---|---|---|---|
| رفتن | رو (row) | رفت (raft) | to go |
| آمدن | آ (ā) | آمد (āmad) | to come |
| بودن | باش (bāsh) | بود (bud) | to be |
| داشتن | دار (dār) | داشت (dāsht) | to have |
| گفتن | گو (gu) | گفت (goft) | to say |
| دیدن | بین (bin) | دید (did) | to see |
| خوردن | خور (khor) | خورد (khord) | to eat |
| خواندن | خوان (khān) | خواند (khānd) | to read |
| نوشتن | نویس (nevis) | نوشت (nevesht) | to write |
| کردن | کن (kon) | کرد (kard) | to do |
| شدن | شو (sho) | شد (shod) | to become |
| گرفتن | گیر (gir) | گرفت (gereft) | to take |

## Negation

| Rule | Example | Translation |
|---|---|---|
| Replace می‌ with نمی‌ | نمی‌روم | I don't go |
| | نمی‌خورم | I don't eat |
| | نمی‌دانم | I don't know |
| | نمی‌فهمم | I don't understand |
| | نمی‌آیم | I'm not coming |

## Ezafe Construction (اضافه)

**Rule: noun + ـِ + adjective/possessive/qualifier**

| Type | Example | Translation |
|---|---|---|
| Noun + Adjective | کتابِ بزرگ | big book |
| Noun + Possessive | کتابِ من | my book |
| Noun + Noun | خانهٔ دوست | friend's house |

**After vowel-ending nouns:** add ـٔ (-ye): خانه → خانهٔ

### Common Ezafe Phrases
| Persian | Transliteration | English |
|---|---|---|
| دوستِ خوب | *dust-e khub* | good friend |
| مادرِ من | *mādar-e man* | my mother |
| رنگِ آبی | *rang-e ābi* | blue color |
| شهرِ بزرگ | *shahr-e bozorg* | big city |
| کتابِ جالب | *ketāb-e jāleb* | interesting book |

## Key Grammar Rules Summary

| Feature | Rule | Example |
|---|---|---|
| Gender | None | کتاب (no gender) |
| Word Order | SOV | من کتاب می‌خوانم |
| "The" | None | کتاب = book or the book |
| "A/An" | noun + ـی or یک + noun | کتابی / یک کتاب |
| Plural | noun + ها (-hā) | کتاب‌ها |
| Adj. Position | After noun (via ezafe) | کتابِ بزرگ |
| Present Tense | می + stem + ending | می‌روم |
| Negation | نمی + stem + ending | نمی‌روم |
| He/She | او (*u*) — one word | او می‌رود |
