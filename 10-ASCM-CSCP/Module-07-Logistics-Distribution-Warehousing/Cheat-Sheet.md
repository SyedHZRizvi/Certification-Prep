## 🚚 5 Transport Modes

| Mode | Speed | Cost | Best For |
|------|-------|------|----------|
| Air | Fastest | Highest | Time-critical, high-value, low-weight |
| Truck | Fast | Medium | Door-to-door domestic |
| Rail | Medium | Low | Heavy bulk, long-distance |
| Ocean | Slowest | Lowest | Global bulky |
| Pipeline | Continuous | Low | Liquids / gases |

**Intermodal** = same container moves across modes.
**Multimodal** = multiple modes under one contract.

## 🏭 Warehouse Operations Flow

```
   Receiving → Putaway → Storage → Picking → Packing → Shipping
                                    ↑
                              Replenishment
```

## 🛒 Picking Strategies

| Method | Best For |
|--------|----------|
| Discrete | Small ops, one order at a time |
| Batch | Many small orders, same zone |
| Zone | Large DC, high SKU (Stock Keeping Unit) count |
| Wave | Synchronize with carrier pickups |
| Goods-to-Person (GTP) | High-throughput e-com |

## 🌐 3PL (Third-Party Logistics) vs 4PL (Fourth-Party Logistics) Quick Reference

| Tier | Description |
|------|-------------|
| 1PL | Self-handled |
| 2PL | Carrier only |
| 3PL | Multi-function outsourced (assets + ops) |
| 4PL / LLP | Orchestrator (no assets) |
| 5PL | Digital marketplace |

## 📐 Logistics KPIs (Key Performance Indicators)

| KPI (Key Performance Indicator) | Definition |
|-----|------------|
| OTD | On-time deliveries / total |
| OTIF (On-Time In-Full) | On-time AND in-full |
| Perfect order | OTIF + undamaged + correct documentation |
| Fill rate | Units shipped / units ordered |
| Order cycle time | Days, order to delivery |
| Cost per unit shipped | Logistics cost / units |

## ⚠️ Anti-Patterns To Recognize

- ❌ Crossdocking everything (only fits predictable, high-velocity)
- ❌ Pure air freight on low-value goods
- ❌ Drop-shipping strategic SKUs needing inventory pooling
- ❌ Treating OTD as OTIF
- ❌ 4PL with asset-heavy operating model
- ❌ Ignoring reverse logistics in design
