# Module 10 Cheat Sheet — Mastery & Fluency (الإتقان والطلاقة)

## Maṣdar (Verbal Noun) Patterns by Form

| Form | Maṣdar Pattern | Example | Meaning |
|------|---------------|---------|---------|
| I | Variable (memorise) | ذَهَاب، كِتَابَة، فَهْم | going, writing, understanding |
| II | تَفْعِيل | تَعْلِيم، تَنْظِيم | teaching, organising |
| III | فِعَال or مُفَاعَلَة | كِفَاح، مُسَاعَدَة | struggle, help |
| IV | إِفْعَال | إِعْلَام، إِرْسَال | informing, sending |
| V | تَفَعُّل | تَعَلُّم، تَكَلُّم | learning, speaking |
| VI | تَفَاعُل | تَعَاوُن، تَبَادُل | cooperation, exchange |
| VII | اِنْفِعَال | اِنْكِسَار، اِنْتِشَار | breaking, spreading |
| VIII | اِفْتِعَال | اجْتِمَاع، اخْتِيَار | meeting, choice |
| X | اسْتِفْعَال | اسْتِغْفَار، اسْتِعْمَال | seeking forgiveness, use |

## Active and Passive Participle Patterns

| Form | Active (فاعل) | Passive (مفعول) |
|------|--------------|---------------|
| I | فَاعِل: كَاتِب (writer) | مَفْعُول: مَكْتُوب (written) |
| II | مُفَعِّل: مُعَلِّم (teacher) | مُفَعَّل: مُعَلَّم (taught) |
| III | مُفَاعِل: مُسَاعِد (helper) | مُفَاعَل: مُسَاعَد (helped) |
| IV | مُفْعِل: مُرْسِل (sender) | مُفْعَل: مُرْسَل (sent) |
| V | مُتَفَعِّل: مُتَعَلِّم (learner) | مُتَفَعَّل: مُتَعَلَّم (learned) |
| X | مُسْتَفْعِل: مُسْتَعْمِل (user) | مُسْتَفْعَل: مُسْتَعْمَل (used) |

## Arabic Rhetoric (البلاغة) — Key Devices

| Device | Arabic | Example |
|--------|--------|---------|
| Simile | تَشْبِيه | الصَّدِيقُ كَالمِرْآة (A friend is like a mirror) |
| Metaphor | اسْتِعَارَة | أَسَدٌ فِي المَعْرَكَة (lion in battle = brave man) |
| Antithesis | طِبَاق | اللَّيْلُ وَالنَّهَار (night and day) |
| Alliteration | جِنَاس | سَالَ السَّيْل (similar sounds) |
| Hyperbole | مُبَالَغَة | رَأَيْتُ البَحْر (I saw an ocean [of generosity]) |
| Euphemism | كِنَايَة | Indirect reference to sensitive matter |

## Conditional Sentences (الشرط)

| Particle | Type | Example |
|---------|------|---------|
| إِنْ + jussive | Simple condition | إِنْ تَدْرُسْ تَنْجَحْ |
| إِذَا + past | More certain | إِذَا دَرَسْتَ نَجَحْتَ |
| لَوْ + past | Contrary-to-fact | لَوْ دَرَسْتَ لَنَجَحْتَ |
| مَنْ | Whoever | مَنْ يَسْعَ يَنَل |

## Relative Pronouns (أسماء الموصول)

| Pronoun | Gender/Number | Example |
|---------|--------------|---------|
| الَّذِي | m. sing. definite | الطَّالِبُ الَّذِي يَدْرُسُ |
| الَّتِي | f. sing. definite | المَرْأَةُ الَّتِي تَعْمَلُ |
| الَّذِينَ | m. pl. definite | الطُّلَّابُ الَّذِينَ نَجَحُوا |
| مَنْ | persons (indefinite) | مَنْ يَعْمَلْ يَنَل |
| مَا | things (indefinite) | مَا تَزْرَعْهُ تَحْصُدْهُ |

## Argumentation Phrases

| Arabic | Function |
|--------|---------|
| يَرَى المُؤَيِّدُون أَنَّ | Proponents argue that |
| فِي المُقَابِل، يَعْتَقِدُ | Conversely, opponents believe |
| مِمَّا لَا شَكَّ فِيه أَنَّ | There is no doubt that |
| وَعَلَى الرَّغْمِ مِنْ ذَلِكَ | Despite this |
| وَيَتَضِحُ مِنْ ذَلِكَ أَنَّ | From this it is clear that |
| وَتَبْقَى المَسْأَلَةُ مَحَلَّ جَدَل | The matter remains controversial |
