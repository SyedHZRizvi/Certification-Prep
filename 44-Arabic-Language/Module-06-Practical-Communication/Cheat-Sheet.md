## Memory Tips for Urdu Speakers

- **Root ك-ت-ب** — writing: كِتَاب (book), مَكْتَب (office), كَاتِب (writer) = Urdu کتاب, مکتب, کاتب
- **Root ش-ر-ك** — partnership: شَرِكَة (company), شَرِيك (partner) = Urdu شریک (sharīk — partner)
- **Root ح-س-ب** — counting: حِسَاب (account), مُحَاسِب (accountant) = Urdu حساب
- **Plural pattern مَفَاعِل**: مَكَاتِب = offices (pl. of مَكْتَب) — this pattern appears with 4-letter words
