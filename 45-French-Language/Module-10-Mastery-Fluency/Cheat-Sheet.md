# Module 10 Cheat Sheet — Maîtrise et Fluidité · C1–C2

## The Four Pillars of Academic French

| Pillar | Principle | Avoid | Use instead |
|---|---|---|---|
| Clarté | Say exactly what you mean | vague filler | precise vocabulary |
| Concision | No redundancy | *il est clair que de toute évidence* | *il est évident que* |
| Cohérence | Logical flow | jumping between points | *premièrement... de plus... enfin...* |
| Élévation | Appropriate register | colloquial words | *par conséquent*, *il convient de* |

## Academic Linking Phrases

| Function | French | English |
|---|---|---|
| Introduce | Il convient d'aborder... | It is appropriate to address... |
| First | Dans un premier temps, | First of all, |
| Second | Dans un second temps, | Secondly, |
| Furthermore | Qui plus est, / De surcroît, | What is more, |
| Concede | Certes,... mais... | Granted,... but... |
| Acknowledge | Force est de constater que... | One must acknowledge that... |
| Challenge | Il serait réducteur de... | It would be reductive to... |
| Deny impossible | On ne saurait nier que... | One cannot deny that... |
| Conclude | En définitive, | Ultimately, |
| Shed light | mettre en lumière | to highlight |
| Challenge | remettre en question | to question/challenge |

## Rhetorical Figures (Figures de Rhétorique)

| Figure | Definition | Example |
|---|---|---|
| Anaphore | Repetition at start of clauses | *Jamais je ne cède. Jamais je ne recule.* |
| Chiasme | Inverted parallel structure | *Manger pour vivre / vivre pour manger* |
| Hyperbole | Exaggeration | *J'ai mille choses à vous dire.* |
| Litote | Understatement | *Ce n'est pas sans mérite.* (= It has real merit) |
| Métaphore | Direct comparison | *La vie est un long fleuve tranquille.* |
| Euphémisme | Softening harsh reality | *Il nous a quittés.* (= He died) |
| Antithèse | Deliberate contrast | *L'union fait la force.* |

## Formal Debate Structure

| Stage | French term | Content |
|---|---|---|
| 1 | Introduction | Contextualise + state thesis |
| 2 | Développement | 2–3 arguments + evidence |
| 3 | Nuance | Concede counterarguments (*Certes... mais...*) |
| 4 | Conclusion + Ouverture | Synthesis + broader question |
