## Answer Key

1. C — Boileau wrote *Ce qui se conçoit bien s'énonce clairement* in his 1674 *Art poétique* (often attributed to Descartes but correctly Boileau).
2. B — Nominalisation = replacing verb phrases with noun phrases for formal register.
3. B — *L'augmentation des prix* is the nominalised form.
4. A — *Dans un premier temps* = in the first place / first of all.
5. B — *Force est de constater* = one must acknowledge / cannot help but notice.
6. B — *Il serait réducteur de* = it would be reductive/oversimplistic to.
7. C — Anaphore = repetition at the start of successive clauses.
8. B — Understatement (*Ce n'est pas sans mérite* = it has real merit) = litote.
9. D — *La vie est un long fleuve tranquille* = a direct comparison without "like/as" = métaphore.
10. C — Inverted parallel structure (*manger pour vivre / vivre pour manger*) = chiasme.
11. B — *En définitive* introduces a concluding/final thought.
12. B — *Qui plus est* = what is more / furthermore.
13. C — *Soutenu / académique* is the highest, most formal register.
14. B — Verlan inverts syllables (from *l'envers* = the reverse).
15. B — *Bizarre* → *zarbi* is the Verlan form.
16. C — *Il nous a quittés* (= he died) = euphémisme.
17. B — *Certes... mais...* = conceding a point while maintaining your overall position.
18. C — *De surcroît* = furthermore / what is more.
19. B — Uses nominalisation (*la décision gouvernementale de modifier*).
20. C — Introduction → Développement → Nuance → Conclusion is the standard French debate structure.
21. B — *Mettre en lumière* = to highlight / shed light on.
22. B — *Il convient d'aborder* = it is appropriate to introduce/raise this question.
23. C — *Remettre en question* = to question / challenge.
24. B — De Gaulle's *Je vous ai compris* is famous for its deliberate ambiguity.
25. C — *On ne saurait nier* uses the conditional (*saurait* = conditional of *savoir*).
26. B — The sentence means the two views complement rather than oppose each other.
27. C — *L'analyse des chercheurs met en évidence* correctly uses nominalisation and formal register.
