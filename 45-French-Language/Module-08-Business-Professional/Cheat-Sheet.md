## Meeting Phrases — Open to Close

| Phase | French |
|---|---|
| Open | Je déclare la séance ouverte. |
| Give floor | Je vous donne la parole. |
| Next point | Passons au point suivant. |
| Propose | Je propose que nous... |
| Agree | Je suis d'accord. |
| Disagree | Je suis contre. / Je ne suis pas d'accord. |
| Vote | Je mets cette proposition aux voix. |
| Close | Je déclare la séance levée. |

## Formal vs Informal Register

| Informal | Formal |
|---|---|
| dire | affirmer / déclarer |
| montrer | démontrer / illustrer |
| finir | achever / mener à terme |
| changer | modifier / apporter des modifications |
| avoir besoin de | nécessiter |
| trouver une solution | résoudre la problématique |

## Key Formal Phrases

| French | English |
|---|---|
| En ce qui concerne... | With regard to... |
| Il convient de souligner que... | It is appropriate to emphasise that... |
| Dans le cadre de notre politique... | Within the framework of our policy... |
| Nous vous prions de bien vouloir... | We kindly request that you... |
| Suite aux décisions prises lors de... | Following the decisions made at... |
| Nous sommes en mesure de vous informer... | We are in a position to inform you... |

## Common False Friends in Business French

| French | Looks like | Actually means |
|---|---|---|
| actuel/actuelle | actual | current / present |
| éventuellement | eventually | possibly |
| un rapport | rapport | a report (also: a relationship) |
| sensible | sensible | sensitive |
| la journée | journey | the day |
