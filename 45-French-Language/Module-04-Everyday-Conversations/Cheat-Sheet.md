# Module 4 — Cheat Sheet: Les Conversations du Quotidien

## Café & Restaurant
| French | English |
|---|---|
| Je voudrais... | I would like... (polite order) |
| Vous désirez? | What would you like? |
| L'addition, s'il vous plaît. | The bill, please. |
| C'est combien? / Ça fait combien? | How much is it? |
| Ça fait... euros. | That comes to... euros. |
| Compris (service compris) | Included (service included) |
| Bon appétit! | Enjoy your meal! |
| C'est délicieux! | It's delicious! |
| Encore un peu de...? | A little more...? |
| Juste un peu, merci. | Just a little, thank you. |
| La carte, s'il vous plaît. | The menu, please. |
| C'est pour emporter. | It's to take away. |

## French Culture Quick Guide
| Topic | Key Fact |
|---|---|
| La bise | Cheek-kiss greeting — 2 kisses standard in most of France |
| Dinner arrival | 10–15 minutes late is polite |
| Dinner gifts | Wine, chocolates, or flowers (NOT chrysanthemums) |
| Lunch time | 12–2 pm (still the main meal in many regions) |
| Dinner time | 7:30–9 pm (later than UK/US) |
| Service compris | Service charge included by law — tipping optional |
| Tipping | Round up or leave small change — not obligatory |
| Phone answering | Give your name / hotel name — never just "allô" |
| *Allô* | Only used by the CALLER to check the line |
