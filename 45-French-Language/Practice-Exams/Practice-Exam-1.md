## Answer Key

```
1.  B — Je m'appelle is the correct reflexive construction for introducing oneself. Tu t'appelles = you're called; il s'appelle = he's called.

2.  A — Merci beaucoup = Thank you very much. S'il te plaît = please; Au revoir = goodbye; Enchanté = nice to meet you.

3.  D — Jaune = yellow. Rouge = red; vert = green; bleu = blue.

4.  C — Huit = 8. Sept = 7; neuf = 9; six = 6.

5.  C — Meeting a superior for the first time requires the formal Bonjour + title + enchanté. Salut and coucou are informal.

6.  D — Bonne nuit = Good night (said when going to bed). Bonsoir = Good evening.

7.  C — Trente-cinq = 35. Quinze = 15; vingt-cinq = 25; quarante-cinq = 45.

8.  B — Ça va bien, merci = I'm fine, thank you. This is the standard positive response to "Comment ça va?"

9.  C — Sœur = sister. Mère = mother; fille = daughter/girl; tante = aunt.

10. D — Vendredi (Friday) follows jeudi (Thursday). The week order is: lundi, mardi, mercredi, jeudi, vendredi, samedi, dimanche.

11. B — Mars = March. Mai = May; avril = April; juin = June.

12. C — La main = the hand. Le pied = foot; la tête = head; le bras = arm.

13. A — Je suis médecin correctly uses être for profession. Avoir is never used to express what you are (profession/nationality).

14. B — Nous avons = we have. Nous sommes = we are; vous êtes = you are; ils/elles ont = they have.

15. C — Je ne parle pas espagnol. Negation in French requires ne...pas around the conjugated verb. Option B (spoken French) is informal and considered incorrect in written/exam contexts.

16. C — J'ai faim = I am hungry (lit. I have hunger). J'ai soif = thirsty; J'ai froid = cold; Je suis fatigué = tired.

17. B — A mixed-gender group (Marie et Pierre = female + male) uses ils (masculine plural) by default in French.

18. C — Le chat est noir = The (specific) cat is black. The definite article le is correct here; un would mean "a" cat.

19. B — J'ai deux frères = I have two brothers. French uses avoir (to have) for family relationships. Sœurs = sisters.

20. A — Elle est grande = She is tall. Adjectives agree with gender; grande is the feminine form of grand.

21. B — Un livre intéressant = an interesting book. Un is the masculine singular indefinite article. Des would require the plural: des livres.

22. C — Comment vous appelez-vous? = What is your name? (formal). Comment allez-vous? = How are you?

23. B — J'ai vingt ans = I am 20 years old. Vingt = 20.

24. D — Les is the plural definite article used before plural nouns (vowel or consonant). L' is only singular (l'ami, l'école).

25. C — Il fait beau = The weather is nice / It is beautiful weather. Il pleut = it rains; il fait froid = it's cold; il fait du vent = it's windy.

26. B — Parlez-vous français? = Do you speak French? Parlez is the vous form. Subject-verb inversion creates the question.

27. D — L'automne = autumn. L'été = summer; le printemps = spring; l'hiver = winter.

28. A — Je viens de France = I come from / am from France. Venir de = to come from.

29. A — Oui, je suis étudiant. The question uses vous êtes (are you), the answer mirrors with je suis (I am). Avoir is never used here.

30. B — À bientôt = See you soon. À demain = see you tomorrow; À tout à l'heure = see you later.
```
