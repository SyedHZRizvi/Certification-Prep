## Argumentation Structure

| Stage | French phrase | Function |
|---|---|---|
| Opening | Tout d'abord, ... | First point |
| Adding | De plus, / En outre, | Add another point |
| Contrasting | Cependant, / Toutefois, | Introduce counter-argument |
| Conceding | Il est vrai que... | Acknowledge opposing view |
| Concluding | En conclusion, / Finalement, | Wrap up |
| Example | Par exemple, / Ainsi, | Illustrate |

## Pronunciation Reminder

- **-tion** endings: pronounced /sjɔ̃/ — *présentation* = /pʁe.zɑ̃.ta.sjɔ̃/
- **-ement** endings: the "e" is often silent — *actuellement* = /ak.tɥɛl.mɑ̃/
- Liaison rule: final consonant links to next word's vowel — *vous‿avez* = /vu.za.ve/
