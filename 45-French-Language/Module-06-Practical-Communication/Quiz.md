## Answer Key

1. C — *Madame, Monsieur,* is the standard formal opening when gender is unknown.
2. B — *Je me permets de vous contacter* = "I am taking the liberty of contacting you."
3. C — The full formal sentence closing is required in French business letters.
4. C — *Cependant* = however.
5. D — *En revanche* = on the other hand (contrast).
6. C — *Assister à* = to attend (false friend — not "to assist").
7. B — French formal closings are a complete sentence, not two words like English.
8. B — *Rester* = to stay/remain (not "to rest" — that's *se reposer*).
9. C — *Antithèse* = acknowledging the opposing view.
10. B — *Par conséquent* = consequently / as a result.
11. C — *À mon avis* = in my opinion.
12. B — *Suite à notre entretien téléphonique* = following our telephone conversation.
13. C — *Le 15 juin 2026* is the correct French formal date format.
14. B — *À quatorze heures trente* = 14:30 = 2:30 pm (24-hour clock used in formal French).
15. C — *Communication* is a true cognate.
16. B — *Librairie* = bookshop (not library; library = *bibliothèque*).
17. C — *Passons maintenant à...* = let us now move on to... (transition).
18. A — *Je voudrais souligner que...* = I would like to emphasise that...
19. A — *Dans l'attente de votre réponse* = awaiting / while waiting for your reply.
20. B — *En effet* = indeed / in fact.
21. C — *Je vous serais reconnaissant(e) de bien vouloir...* is the formal polite request.
22. B — *Monsieur,* is the standard formal address for an unknown man.
23. B — *Toutefois* is a synonym of *cependant* (however).
24. C — *N'hésitez pas à me contacter* = do not hesitate to contact me.
25. B — *Par exemple* = for example.
26. B — *Actuellement* = currently (false friend — NOT "actually" which is *en fait*).
27. B — *Je vous remercie de votre attention* = thank you for your attention.
