# Module 7 Cheat Sheet — Grammaire Avancée · B2

## Present Subjunctive Formation

**Rule:** ils-form present tense → remove -ent → add endings: -e, -es, -e, -ions, -iez, -ent

| Infinitive | ils form | Subjonctif stem | que je |
|---|---|---|---|
| parler | parlent | parl- | parle |
| finir | finissent | finiss- | finisse |
| vendre | vendent | vend- | vende |
| prendre | prennent | prenn- | prenne |

## Subjunctive Trigger Phrases

| Always subjunctive | Meaning |
|---|---|
| il faut que | it is necessary that |
| il est possible que | it is possible that |
| bien que / quoique | although |
| pour que / afin que | so that |
| avant que | before |
| à moins que | unless |
| vouloir que | to want (sb) to |
| avoir peur que | to be afraid that |
| être content(e) que | to be happy that |
| douter que | to doubt that |

| Never subjunctive (indicative) | Meaning |
|---|---|
| je pense que | I think that |
| il est certain que | it is certain that |
| il est évident que | it is obvious that |
| je sais que | I know that |

## Si Clause System

| Time reference | Si clause tense | Result clause tense |
|---|---|---|
| Real/open condition | présent | futur simple |
| Hypothetical present | imparfait | conditionnel présent |
| Hypothetical past | plus-que-parfait | conditionnel passé |

**Examples:**
- *Si tu travailles, tu réussiras.* (real)
- *Si tu travaillais, tu réussirais.* (hypothetical present)
- *Si tu avais travaillé, tu aurais réussi.* (hypothetical past)

## Conditionnel Passé Formation

**avoir/être (conditional) + past participle**

| Pronoun | Avoir | Être |
|---|---|---|
| je / j' | j'aurais | je serais |
| tu | tu aurais | tu serais |
| il/elle | il aurait | il serait |
| nous | nous aurions | nous serions |
| vous | vous auriez | vous seriez |
| ils/elles | ils auraient | ils seraient |

## Reported Speech Tense Shifts

| Direct speech | → | Reported speech |
|---|---|---|
| présent | → | imparfait |
| passé composé | → | plus-que-parfait |
| futur simple | → | conditionnel présent |
| futur proche | → | allait + infinitif |
| impératif | → | de + infinitif |

## Plus-que-parfait Formation

**imparfait of avoir/être + past participle**

*j'avais mangé / tu avais mangé / il avait mangé / nous avions mangé / vous aviez mangé / ils avaient mangé*

**Use:** action completed BEFORE another past action.
- *Quand je suis arrivé, elle **était déjà partie**.* (She had already left when I arrived.)
