## Answer Key

1. C — *qu'il vienne* is the correct present subjunctive of *venir*.
2. C — *que je sois* is the subjunctive of *être* (irregular).
3. C — *Il faut que* always triggers the subjunctive; the others take the indicative.
4. B — *aies* is the subjunctive of *avoir* for *tu*.
5. D — *qu'ils aillent* is the irregular subjunctive of *aller*.
6. B — *il aurait parlé* = conditional of *avoir* + past participle.
7. B — Si + plus-que-parfait + conditionnel passé = hypothetical past.
8. C — Si + présent → futur simple is the real/open condition.
9. C — *Tu aurais pu me prévenir!* = reproach (you could have warned me).
10. C — Présent becomes imparfait in past reported speech.
11. C — *viendrait* is the reported form of *viendrai* (futur → conditionnel).
12. C — *aujourd'hui* → *ce jour-là* in reported speech.
13. C — Plus-que-parfait = action before another past action.
14. C — *était partie* = plus-que-parfait (être + past participle).
15. C — *avaient mangé* = plus-que-parfait with *avoir*.
16. C — *Bien que* always requires the subjunctive.
17. B — *Pour que* (so that) always triggers the subjunctive.
18. B — *que nous fassions* = irregular subjunctive of *faire*.
19. B — Imperative becomes *de + infinitif* in reported speech.
20. D — Imparfait does NOT change to passé composé; it stays as imparfait in reported speech.
21. B — Si + imparfait + conditionnel présent = hypothetical present/future.
22. C — *demain* → *le lendemain* in reported speech.
23. B — *J'aurais voulu* = I would have liked (past unrealised wish).
24. C — *Avant que* always requires the subjunctive; A, B, D take the indicative.
25. B — *demander si* introduces an indirect yes/no question.
