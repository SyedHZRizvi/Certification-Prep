# Module 2 — Cheat Sheet: Le Vocabulaire Essentiel

## Gender Quick Guide
| Usually Masculine | Usually Feminine |
|---|---|
| endings: -age, -eau, -isme, -ment | endings: -tion, -té, -ure, -ence, -ance |
| *le voyage, le bateau, le tourisme* | *la nation, la liberté, la culture* |

**Golden Rule:** Learn every noun WITH its article: *le pain, la maison, l'ami, les enfants*
