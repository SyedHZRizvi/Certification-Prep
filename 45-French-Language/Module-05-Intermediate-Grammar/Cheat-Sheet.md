# Module 5 — Cheat Sheet: La Grammaire Intermédiaire

## Passé Composé Quick Reference

### Forming Past Participles
| Verb Type | Rule | Example |
|---|---|---|
| -ER verbs | remove -er → **-é** | parler → **parlé** |
| -IR verbs | remove -ir → **-i** | finir → **fini** |
| -RE verbs | remove -re → **-u** | vendre → **vendu** |

### Key Irregular Past Participles
| Verb | PP | | Verb | PP |
|---|---|---|---|---|
| avoir | eu | | vouloir | voulu |
| être | été | | venir | venu |
| faire | fait | | mettre | mis |
| prendre | pris | | dire | dit |
| voir | vu | | écrire | écrit |
| savoir | su | | lire | lu |
| pouvoir | pu | | boire | bu |
| naître | né | | mourir | mort |
| ouvrir | ouvert | | recevoir | reçu |

### DR MRS VANDERTRAMP (être verbs — past participle agrees with subject!)
Devenir · Revenir · Mourir · Rester · Sortir · Venir · **Aller** · Naître · Descendre · Entrer · Rentrer · Tomber · Retourner · **Arriver** · Monter · Partir

Agreement: *Il est arrivé / Elle est arrivée / Ils sont arrivés / Elles sont arrivées*

## Imparfait Conjugation

**Formula:** nous form (present) → remove -ons → add endings

| Pronoun | Ending | Parler | Finir | Être* |
|---|---|---|---|---|
| je | **-ais** | parlais | finissais | étais |
| tu | **-ais** | parlais | finissais | étais |
| il/elle | **-ait** | parlait | finissait | était |
| nous | **-ions** | parlions | finissions | étions |
| vous | **-iez** | parliez | finissiez | étiez |
| ils/elles | **-aient** | parlaient | finissaient | étaient |

*Être is the only verb with an irregular imparfait stem: **ét-**

## Passé Composé vs Imparfait — Decision Guide

| Passé Composé | Imparfait |
|---|---|
| Completed action | Ongoing / habitual action |
| Happened at specific time | General state or feeling |
| Sudden event (soudain, tout à coup) | Background description |
| Specific number of times | Used to / would |
| Plot point in a story | Scene-setting in a story |

**Marker words:**
- PC: soudain, tout à coup, un jour, hier, à midi, trois fois
- Imparfait: tous les jours, chaque semaine, toujours, quand j'étais enfant, pendant longtemps

## Future Tenses

### Futur Proche = aller (present) + infinitive
*je vais parler / tu vas parler / il va parler / nous allons parler / vous allez parler / ils vont parler*

### Futur Simple Endings (add to infinitive; -RE verbs drop final e)
**-ai / -as / -a / -ons / -ez / -ont**

## Reflexive Verbs

**Structure:** Subject + reflexive pronoun + verb
me (m') · te (t') · se (s') · nous · vous · se (s')

| French | English |
|---|---|
| se lever | to get up |
| se coucher | to go to bed |
| se réveiller | to wake up |
| se laver | to wash oneself |
| se doucher | to shower |
| se brosser les dents | to brush one's teeth |
| s'habiller | to get dressed |
| s'appeler | to be called |
| se dépêcher | to hurry |
| se reposer | to rest |

**Passé composé:** Reflexive verbs always use *être* (past participle agrees with subject):
*Elle s'est levée.* / *Ils se sont habillés.*

## Object Pronouns

| Pronoun | Meaning | Position |
|---|---|---|
| le / la / les | him·her·it / them (direct) | before conjugated verb |
| lui / leur | to him·her / to them (indirect) | before conjugated verb |
| me / te / nous / vous | me·you·us·you | before conjugated verb |
| y | there / to it | before conjugated verb |
| en | of it / some / from there | before conjugated verb |

Before infinitive: pronoun goes before the **infinitive**.
*Je veux **le** manger.* (I want to eat it.)

---

## Depuis (Since / For — Ongoing Actions)

**Present tense + depuis = have been doing**
*J'habite ici depuis 2020.* = I have been living here since 2020.
*Il travaille depuis trois ans.* = He has been working for three years.

## Comparatives and Superlatives

| Type | French | Example |
|---|---|---|
| More than | plus...que | *Il est plus grand que moi.* |
| Less than | moins...que | *C'est moins cher qu'avant.* |
| As...as | aussi...que | *Elle est aussi intelligente que lui.* |
| The most | le/la/les plus + adj | *C'est la plus belle ville.* |
| The least | le/la/les moins + adj | *C'est le moins cher.* |

**Irregulars:**
| Adjective | Comparative | Superlative |
|---|---|---|
| bon (good) | meilleur | le meilleur (the best) |
| mauvais (bad) | pire | le pire (the worst) |
| bien (well) | mieux | le mieux (the best — adverb) |
