# Module 3 — Cheat Sheet: La Grammaire de Base

## ÊTRE (To Be) — Fully Conjugated

| Pronoun | Être | English |
|---|---|---|
| je | **suis** | I am |
| tu | **es** | you are |
| il / elle / on | **est** | he/she/it is / we are |
| nous | **sommes** | we are |
| vous | **êtes** | you are |
| ils / elles | **sont** | they are |

Key uses: identity, nationality, profession (no article!), location, descriptions
*Je suis médecin. Elle est française. Nous sommes à Paris.*

---

## AVOIR (To Have) — Fully Conjugated

| Pronoun | Avoir | English |
|---|---|---|
| je | **ai** | I have |
| tu | **as** | you have |
| il / elle / on | **a** | he/she/it has |
| nous | **avons** | we have |
| vous | **avez** | you have |
| ils / elles | **ont** | they have |

**Avoir idioms (use AVOIR, not être):**
- *avoir faim* = to be hungry
- *avoir soif* = to be thirsty
- *avoir chaud/froid* = to be hot/cold
- *avoir peur* = to be afraid
- *avoir raison/tort* = to be right/wrong
- *avoir X ans* = to be X years old
- *avoir besoin de* = to need
- *avoir envie de* = to want/feel like
- *avoir l'air* = to seem/look like

## Regular -ER Verb Conjugation: Parler Pattern

| Pronoun | Ending | Parler | Aimer | Habiter |
|---|---|---|---|---|
| je | **-e** | parle | aime | habite |
| tu | **-es** | parles | aimes | habites |
| il/elle/on | **-e** | parle | aime | habite |
| nous | **-ons** | parlons | aimons | habitons |
| vous | **-ez** | parlez | aimez | habitez |
| ils/elles | **-ent** | parlent | aiment | habitent |

Note: *manger → mangeons* (nous) — extra e for soft G

## Articles Chart

| | Definite (the) | Indefinite (a/some) |
|---|---|---|
| Masc. singular | **le** | **un** |
| Fem. singular | **la** | **une** |
| Before vowel/h | **l'** | *same — un/une* |
| All plural | **les** | **des** |
| After negation | *unchanged* | → **de / d'** |

## Adjective Agreement

| | Masculine | Feminine |
|---|---|---|
| Singular | (base form) | + **e** |
| Plural | + **s** | + **es** |

**Irregular pairs:**
| Masc | Fem | Meaning |
|---|---|---|
| bon | bonne | good |
| mauvais | mauvaise | bad |
| heureux | heureuse | happy |
| beau (bel) | belle | beautiful |
| nouveau (nouvel) | nouvelle | new |
| vieux (vieil) | vieille | old |
| long | longue | long |
| blanc | blanche | white |

**Position — BAGS before noun:**
Beauty: *beau, joli* | Age: *jeune, vieux* | Goodness: *bon, mauvais* | Size: *grand, petit*
All others go AFTER the noun.

## Negation: The Sandwich Rule

**Subject + ne + [conjugated verb] + pas**

| Negative Word | Meaning | Example |
|---|---|---|
| ne...pas | not | Je ne parle pas anglais. |
| ne...jamais | never | Je ne mange jamais de viande. |
| ne...rien | nothing | Il ne dit rien. |
| ne...personne | nobody | Je ne vois personne. |
| ne...plus | no longer | Elle ne travaille plus ici. |
| ne...que | only | Je n'ai que dix euros. |

After negation: un/une/des → **de (d')**

## Subject Pronouns Quick Reference

| French | English | Notes |
|---|---|---|
| je / j' | I | j' before vowel |
| tu | you (informal) | friends, family |
| il / elle / on | he / she / one/we | *on* = informal we |
| nous | we | formal |
| vous | you (formal/plural) | always formal singular |
| ils / elles | they (m) / they (f) | *ils* for any mixed group |

## Question Formation Summary

| Method | Example | Use |
|---|---|---|
| Intonation ↑ | *Tu parles français?* | Informal speech |
| Est-ce que | *Est-ce que tu parles français?* | All situations |
| Inversion | *Parlez-vous français?* | Formal / written |

**Question words:** Qui (who) · Que/Qu'est-ce que (what) · Où (where) · Quand (when) · Comment (how) · Pourquoi (why) · Combien (how much/many) · Quel/Quelle (which/what)
