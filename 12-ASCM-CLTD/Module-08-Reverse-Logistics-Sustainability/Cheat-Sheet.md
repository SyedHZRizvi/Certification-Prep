## 🔄 5-Step Returns Process

```
1. Authorization (RMA)
2. Receipt at returns center
3. Gatekeeping / disposition
4. Refurbish / repackage
5. Recovery (resell, liquidate, recycle, dispose)
```

🧠 Gatekeeping = the most leveraged step.

---

## ♻️ Disposition Options

| Option | When |
|--------|------|
| Restock as new | Unopened, undamaged |
| Open-box | Lightly used, functional |
| Refurbish | Clean, test, repackage |
| Remanufacture | Disassemble, restore, new warranty |
| Liquidate | Bulk sale to wholesaler |
| Donate | Charitable + tax benefit |
| Recycle | Reclaim materials |
| Hazardous disposal | Regulated waste |
| Destroy | Counterfeit, recalled |

---

## 🔁 Refurbish vs Remanufacture vs Recycle

| Term | What |
|------|------|
| Refurbish | Light: clean, test, repackage; limited warranty |
| Remanufacture | Heavy: disassemble, restore, new-equivalent warranty |
| Recycle | Destroy form; reclaim raw materials |
| Repurpose | Upcycle for different use |

---

## 🌳 Circular Economy — 7Rs

```
Refuse → Reduce → Reuse → Refurbish/Remanufacture/Recycle → Recover → Dispose
```

## 📊 GHG Protocol Scopes

| Scope | Examples |
|-------|----------|
| Scope 1 | Your fleet's tailpipes, your boilers |
| Scope 2 | Grid electricity for your warehouse |
| Scope 3 | Purchased transportation (carriers), suppliers, customer use |

🚨 Purchased carrier freight = Scope 3, not Scope 1.

---

## 🌐 Emissions by Mode (g CO₂e per ton-km)

| Mode | Range |
|------|-------|
| Air | 500–1,500 |
| Truck (medium) | 200+ |
| Truck (heavy) | 60–100 |
| Rail | 18–30 |
| Ocean (container) | 8–18 |
| Pipeline | 5–10 |

🧠 Air is 50–100× more carbon-intensive than ocean.

## 📊 Sustainability KPIs

- Carbon per shipment / per unit
- Empty miles %
- Backhaul rate
- Mode mix % (air vs ocean vs rail)
- EV / alt-fuel % of fleet
- Diversion rate (waste from landfill)
- Returns recovery value $
- Energy intensity (kWh/unit)

---

## 🏆 Exam Power Phrases

Usually CORRECT:
- ✅ "Mode shift is the highest-leverage emissions lever"
- ✅ "Gatekeeping is the most critical returns step"
- ✅ "Closed-loop = your products cycle back to you"
- ✅ "Remanufactured comes with new-equivalent warranty"
- ✅ "Purchased transport is Scope 3"

Usually WRONG:
- ❌ "Offsets fix emissions"
- ❌ "Recycling = reuse"
- ❌ "Refurbished = remanufactured"
- ❌ "Reverse logistics is just returns"
- ❌ "Purchased carriers = Scope 1"

---

## ⚠️ Anti-Patterns

- ❌ Free returns with no gatekeeping
- ❌ Treating returns as a finance write-off only
- ❌ Mixing ISO 14001 (EMS) with ISO 14064 (GHG)
- ❌ Air-freight default without sustainability check
