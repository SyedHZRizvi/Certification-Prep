# 🚛 ASCM CLTD Track (Certified in Logistics, Transportation and Distribution)

> **Goal:** Pass the ASCM CLTD exam with a scaled score of 300+ (≈ 70% raw)
> **Duration:** 12–16 weeks part-time
> **Cost:** $695 (ASCM member) / $1,095 (non-member) — exam fee only

---

## 🎯 What You'll Learn

By the end of this track, you'll be able to:
- Design a logistics network — greenfield/brownfield, hub-and-spoke vs point-to-point, center-of-gravity siting
- Plan capacity for warehouses, fleets, and DCs across seasonal and slow/fast-moving SKUs
- Run an order-to-cash cycle with ATP/CTP, allocation, wave/zone/batch picking, and perfect-order metrics
- Configure DRP (Distribution Requirements Planning) for multi-echelon inventory with postponement and push/pull replenishment
- Operate a modern warehouse — receive, put-away, slot, pick, pack, ship — with WMS, AS/RS, AGV, voice and pick-to-light
- Choose the right transportation mode (FTL, LTL, intermodal, rail, ocean FCL/LCL, air, parcel, pipeline) and TMS strategy
- Apply Incoterms 2020 (all 11), customs/HS basics, FTZs, USMCA/EU/CPTPP and global compliance
- Build reverse logistics, returns, refurbish/remanufacture/recycle, ISO 14001 and green-logistics programs

## 🎓 What Is The CLTD Exam?

| Detail | Specification |
|--------|---------------|
| Provider | ASCM (Association for Supply Chain Management, formerly APICS) |
| Cost | $695 (ASCM member) / $1,095 (non-member) |
| Time | 3.5 hours (210 minutes) |
| Questions | 150 multiple-choice (130 scored, 20 pretest — you can't tell which) |
| Pass mark | **300+ scaled score** (200–350 scale, ≈ 70% raw) |
| Format | Computer-based at Pearson VUE testing centers or OnVUE online proctored |
| Validity | **5 years** — maintained via 75 maintenance points |
| Retake | Up to 3 attempts per ATT (Authorization To Test) window |
| Prereqs | None formally — recommended 3+ years logistics experience |

## ⚠️ The 7 Most Common Reasons People Fail

1. ❌ Studied **Incoterms 2010** (DAT) instead of **Incoterms 2020** (DPU replaces DAT, and DDP/DAP responsibilities shifted)
2. ❌ Confused **FOB** in US domestic usage with **FOB** under Incoterms 2020 (FOB is OCEAN/INLAND-WATERWAY ONLY — never use FOB for a container shipped by truck/air)
3. ❌ Memorized definitions but couldn't apply mode-selection trade-offs to scenario questions
4. ❌ Underestimated the **global** scope — half the exam is non-US (HS codes, customs, FX, regional FTAs)
5. ❌ Skipped sustainability/reverse — it is only ~9% but easy points if studied
6. ❌ Didn't time-budget — 3.5 hours / 150 questions = **84 seconds per question**; ran out of time on the last 20
7. ❌ Treated **DRP** like MRP — they look similar but DRP pulls from DCs upstream, MRP explodes BOMs downstream
