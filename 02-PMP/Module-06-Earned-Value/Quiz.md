## 🎯 Answers

### Q1: B (EV - AC)
### Q2: A (EV / PV)
### Q3: C (CPI = $0.85 earned per $1 spent — over budget)
### Q4: B (>1 = ahead)
### Q5: B (BAC / CPI)
### Q6: B (EAC - AC = remaining work cost)
### Q7: A (BAC - EAC)
### Q8: B (CPI = EV/AC = 60/80 = 0.75)
### Q9: B (SV = EV - PV = 50 - 40 = +10K)
### Q10: B (EAC = 500/0.8 = $625K)
### Q11: B (both <1 = bad)
### Q12: B (>1 = harder than past pace)
### Q13: **A. -$30K**
CV = EV - AC = $120K - $150K = -$30K. Negative CV indicates the project is over budget.
### Q14: **B. 0.80**
CPI = EV / AC = $120K / $150K = 0.80. The team earns only $0.80 of value for every $1.00 spent — cost-inefficient.
### Q15: **B. $500K**
EAC (default) = BAC / CPI = $600K / 1.20 = $500K. Because CPI > 1, the project is forecast to finish under budget.
### Q16: **B. 0.80**
SPI = EV / PV = $200K / $250K = 0.80. The project is progressing at only 80% of the planned schedule pace — behind schedule.
### Q17: **B. -$200K**
VAC = BAC - EAC = $800K - $1,000K = -$200K. A negative VAC means the project is forecast to finish $200K over budget.
### Q18: **B. $450K**
ETC = EAC - AC = $750K - $300K = $450K. That is how much more is expected to be spent to complete the work.
### Q19: **B. 1.20**
TCPI (to BAC) = (BAC - EV) / (BAC - AC) = ($500K - $200K) / ($500K - $250K) = $300K / $250K = 1.20. Future work must be 20% more efficient than past performance to finish at the original budget.
### Q20: **B. Ahead of schedule, over budget**
SPI > 1 means ahead of schedule; CPI < 1 means over budget. The interpretation rules: positive variance / >1 index = good for that dimension only.
### Q21: **B. EAC = AC + (BAC - EV)**
This formula assumes the cost variance to date is a one-time anomaly and remaining work will follow the original budgeted rate. The default BAC/CPI assumes current efficiency continues.
### Q22: **B. Behind schedule and over budget**
SV = EV - PV = $400K - $500K = -$100K (behind). CV = EV - AC = $400K - $500K = -$100K (over). Both negative variances mean the project is performing poorly on both dimensions.
### Q23: **C. Under budget by $15K**
CV is a cost variance (EV - AC). Positive = under budget. Schedule variance would be SV, not CV. Memory aid: Cost involves AC; Schedule involves PV.
### Q24: **A. -$62.5K**
CPI = EV/AC = $100K/$125K = 0.80. EAC = BAC/CPI = $250K/0.80 = $312.5K. VAC = BAC - EAC = $250K - $312.5K = -$62.5K (will finish $62.5K over budget).
