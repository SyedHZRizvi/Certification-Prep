# 📋 PMP Module 4 Cheat Sheet: Business Environment (8% of exam)

---

## 💰 Project Selection Metrics

| Metric | Higher = Better? |
|--------|-------------------|
| NPV | ✅ Yes |
| IRR | ✅ Yes |
| BCR | ✅ Yes (>1 profitable) |
| ROI | ✅ Yes |
| Payback | ❌ Lower = better |

---

## 🌐 Business Case
Justifies the project. Includes: strategic alignment, costs, benefits, ROI, alternatives.

## 🎯 Benefits Realization
- Often AFTER project completion
- Tracked via Benefits Management Plan
- Multiple horizons

---

## ⚖️ Compliance Types
- 🏛️ Regulatory (laws)
- 🏢 Organizational (policies)
- 🌍 Standards (ISO, IEEE)
- 🔐 Security

🚨 Non-compliance = fines, shutdown, reputation, personal liability

---

## 🔄 OCM Models

### Lewin
Unfreeze → Change → Refreeze

### Kotter (8 steps)
1. Create urgency
2. Build coalition
3. Form vision
4. Communicate vision
5. Empower action
6. Short-term wins
7. Sustain acceleration
8. Anchor in culture

### ADKAR
**A**wareness → **D**esire → **K**nowledge → **A**bility → **R**einforcement
