B. Servant leader
C. Auditor
D. Director

### Q7. Hybrid approach is best when:
A. Always
B. Different parts have different certainty levels
C. The team prefers it
D. Compliance requires it

### Q8. Pair programming and TDD are practices from:
A. Scrum
B. Kanban
C. XP (Extreme Programming)
D. Lean

### Q9. Cycle time measures:
A. Sprint length
B. Time from start to done for a work item
C. Time between releases
D. Hours worked

B. Bottlenecks in the workflow
