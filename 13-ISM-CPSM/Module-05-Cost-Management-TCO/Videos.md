## ✅ After watching

1. Define TCO and list the 3 phases.
2. Run a should-cost build-up on a paperclip and a laptop. Which had more cost components?
3. Distinguish value analysis from value engineering.
4. Compute the unit cost at unit #16 on an 80% learning curve starting at $100/unit.
5. When does hedging make sense? When is it speculation?
