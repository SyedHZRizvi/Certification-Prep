## 🪜 Kotter's 8 Steps (MEMORIZE)

1. Create urgency
2. Form a coalition
3. Create vision
4. Communicate the vision
5. Remove obstacles
6. Create short-term wins
7. Build on the change
8. Anchor in culture

🧠 **U-CO-VI-COM-RE-WIN-BU-AN**

## 👥 Tuckman 5 Stages

| Stage | Leader Role |
|---|---|
| Forming | Set direction |
| Storming | Coach through conflict |
| Norming | Reinforce norms |
| Performing | Get out of the way |
| Adjourning | Celebrate + capture |

## ⏱️ Estimation

| Method | When |
|---|---|
| Expert judgment | Quick |
| Analogous | No data, similar past |
| Parametric | Statistical model |
| PERT 3-point | (O + 4M + P) / 6 |
| Bottom-up | Most accurate |

---

## 🎯 Critical Path

Longest dependent sequence = minimum duration.
Crash CRITICAL path tasks to compress.
Float on non-critical = slip tolerance.

---

## 📊 Balanced Procurement KPIs

| Category | Example |
|---|---|
| Financial | Savings $ · SUM · working capital |
| Quality | PPM defects |
| Delivery | OTD % · perfect order rate |
| Operational | Maverick % · sourcing cycle time |
| Strategic | % w/ category strategy |
| People | Engagement · diversity |
| Risk | High-risk supplier % |
| ESG | Scope 3 carbon · diverse spend |

**SUM world-class = 80%+**

🧠 KPIs must be **SMART**.

## 🏆 Exam Power Phrases

- ✅ "Diagnose ADKAR gap before intervention"
- ✅ "Stakeholder analysis first"
- ✅ "Influence with data + social proof + relationship"
- ✅ "Cross-functional team for complex outcomes"
- ✅ "Crash the CRITICAL path"
- ✅ "SMART, balanced KPIs"

## ⚠️ Usually Wrong

- ✗ "Mandate from above unilaterally"
- ✗ "Skip stakeholder analysis"
- ✗ "Cost is the only KPI"
- ✗ "Crash any task to save time"
- ✗ "Storming = bad team"
- ✗ "Smooth over an audit finding"
