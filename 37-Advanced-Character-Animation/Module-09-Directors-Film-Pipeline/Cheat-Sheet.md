# Module 9 Cheat Sheet — Directors, Notes, and the Film Pipeline 🎬

## Decoding Director's Notes

| Director Says | Usually Means | Technical Fix |
|--------------|--------------|--------------|
| "Too much" | Over-animation | Reduce secondary; simplify performance |
| "Not alive" | Under-animation | Add breathing, eye movement, timing variety |
| "Reading wrong" | Wrong intent | Clarify objective; fresh-eyes review |
| "More energy" | Too slow | Sharpen anticipations; tighten spacing |
| "It feels off" | Unspecified | ASK: "Can you point to a specific moment?" |
| "Almost there" | 1–2 things wrong | Show again; ask which moment specifically |

**The one clarifying question:** *"Can you point to a specific moment in the shot that isn't working?"*

---

## Revision Etiquette — 4 Rules

```
1. NEVER argue in the review room
2. IMPLEMENT the exact note FIRST (even if you disagree)
3. Show the SAME shot from the SAME angle (only change what was noted)
4. COMMUNICATE proactively — flag delays before they become surprises
```

---

## "Yes, And" Approach

```
YES:  Implement the exact note as stated
AND:  Optionally offer one variation that builds on it

"I implemented the change you asked for — I also have a 
 take with faster timing if you'd like to compare."
```

---

## Shot Approval Chain

```
1. ANIMATION SUPERVISOR   → Technical quality + character consistency
2. DIRECTOR OF ANIMATION  → Performance + cross-scene consistency  
3. DIRECTOR               → Story performance + emotional intent
4. PRODUCER (select shots)→ Screen time + budget impact

Conflicting notes between levels → Escalate to Director of Animation
```

**"Approved" ≠ guaranteed in film. Editorial can still cut the shot.**

## Feature Film Feedback Loop

| Stage | Duration |
|-------|---------|
| Assignment (Anim Sup + animator) | 30 min |
| Blocking pass | 2–5 days |
| Blocking review | 30 min |
| Spline pass | 3–7 days |
| First spline review (full team + director) | 15–30 min |
| Revision pass | 1–3 days |
| Final review | 15 min |
| Approval | — |

---

## Animating to Editorial

```
BEFORE BLOCKING:
  ✓ Watch editorial cut 5 times
  ✓ Note cut-in and cut-out frames
  ✓ Confirm length with editorial team
  ✓ Know preceding shot's emotional delivery
  ✓ Know next shot's emotional requirement

HANDLES: Add 4–8f held pose at start AND end of every shot
         Never deliver a shot starting/ending on a motion extreme
```

---

## Working with Your Animation Supervisor

| Do | Don't |
|----|-------|
| Show work early and often | Surprise in director review |
| Flag schedule problems immediately | Hide a late shot |
| Accept technical notes without argument | Defend your choices |
| Offer variations when you have them | Implement changes without communicating |

---

## Key Pipeline Terms

| Term | Definition |
|------|-----------|
| Dailies | Group review session; all animators present work |
| Handles | 4–8f of hold at shot start/end |
| Approved | Director signed off; technically/creatively complete |
| ShotGrid | Industry-standard shot tracking software |
| Anim Sup | Animation Supervisor — your primary pipeline contact |
