# Module 10 Cheat Sheet — The Demo Reel 🎥

## The Three Laws of Shot Selection

```
LAW 1:  Reel = weakest shot. Every shot is watched.
LAW 2:  Show what you want to get hired to do.
LAW 3:  Variety — dialogue + mechanics + emotion + creature + complex shot.
```

---

## Shot Ordering Formula

```
POSITION 1:    Your SINGLE BEST shot (hook immediately)
POSITION 2:    Strong, DIFFERENT type (proves versatility)
POSITIONS 3-5: Range of competencies (invest the recruiter)
2ND TO LAST:   High-energy or physical shot (re-energize)
LAST:          Strong technical or emotional closer

NEVER: Put weakest first / Front-load then taper / Two similar shots back-to-back / End weak
```

---

## Length Standards

| Length | Status |
|--------|--------|
| Under 60 seconds | Too short — limited range |
| 60–74 seconds | Acceptable |
| 75–85 seconds | Sweet spot |
| 86–90 seconds | Acceptable |
| Over 90 seconds | Tests patience — trim |

**Time per shot:** 3–5s (action/secondary) · 5–8s (mechanics) · 8–12s (dialogue/performance) · 12–15s (complex/creature) · 15+s (max once, extraordinary only)

---

## Music — 4 Rules

```
1. SUPPORT, don't perform (music ≠ the performance)
2. TEMPO matches reel pacing
3. NO commercial hits (Bohemian Rhapsody problem)
4. CUT to musical structure (transitions align with beats)

Use: Royalty-free orchestral (Epidemic Sound, Artlist)
Avoid: Famous songs / friend's band / licensed pop
```

## Peer Review Protocol

```
1. Find 2 honest peers (not close friends)
2. Show reel once — watch THEIR FACES, not screen
3. Ask exactly TWO questions:
   → "Which shot made you most engaged?"
   → "Was there a moment you wanted to look away?"
4. Show a second time: "Call out any moment of confusion"
5. Write everything down. NEVER defend.

Rule: Both reviewers ID same shot as weak → CUT IT
```

## Pre-Submission Checklist

- [ ] Opens with single strongest shot
- [ ] 60–90 seconds total
- [ ] Title card: name + email + portfolio URL
- [ ] NO home address
- [ ] Music below dialogue volume in dialogue shots
- [ ] Music doesn't end abruptly or loop
- [ ] 1080p minimum export
- [ ] Tested on laptop screen (not just large monitor)
- [ ] Two peer reviews completed
- [ ] All work is original or clearly credited

---

## Self-Editing Test

```
Play reel with AUDIO OFF.
For each shot ask: "Does this earn its time?"
If you think "this part is slow" → cut or shorten that shot.
```

**First 5 seconds test:** Watch a new viewer's eyes during opening. Leaning back = change the opener.

## Three-Pass Self-Critique Protocol

### Pass 1: Technical (audio OFF)
```
[ ] COG traceable through every shot?
[ ] Contact frames have correct ground reaction?
[ ] Secondary elements correctly offset?
[ ] No pops or snaps between shots?
[ ] Quality consistent or dropping mid-reel?
[ ] Starts and ends on clear pose (not motion)?
```

### Pass 2: Performance (audio ON)
```
[ ] Character's objective clear in first 3 seconds?
[ ] At least one unexpected performance choice per shot?
[ ] Eyes tell same story as body?
[ ] Subtext visible without knowing the script?
```

### Pass 3: Editorial (show to stranger)
```
[ ] Does opening shot earn continued watching in 3–5 seconds?
[ ] Can stranger describe what each character wants?
[ ] Any moment where stranger's attention drifts?
[ ] Does reel feel longer than 90 seconds? (if yes: cut)
```
