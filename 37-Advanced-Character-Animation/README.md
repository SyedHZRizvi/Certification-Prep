# Advanced Character Animation, Acting-Level Character Performance

> **The course that trains you to the performance standard studios like Pixar, DreamWorks, Riot, and Framestore hire at.**

This is the professional-grade character animation curriculum covering every discipline you need to work at the top tier of the industry. It is built to the same pedagogical standard as the site's engineering and cloud courses, deep, specific, and career-oriented.

---

## What You Will Learn

| Module | Topic | Career Relevance |
|--------|-------|-----------------|
| 01 | Acting and Performance | Stanislavski, Meisner, video reference strategy, internal monologue |
| 02 | Body Mechanics and Weight | COG, punch/jump/fall mechanics, real vs. animated physics |
| 03 | Facial Animation | FACS, 7 universal expressions, dead eye fix, brow mastery |
| 04 | Advanced Lip Sync | Preston Blair, 2-frame lead, less is more, multilingual challenges |
| 05 | Creature Animation | Quadruped gaits, bird flight, fish locomotion, behavioral borrowing |
| 06 | Crowd Simulation | Massive, Houdini CROWDS, sim vs. keyframe, behavioral layers |
| 07 | Motion Capture | Optical/inertial/markerless, cleanup pipeline, MotionBuilder, when NOT to use mocap |
| 08 | Graph Editor Advanced | Reading curves, tangent mastery, S-curve, stepped workflow, expressions |
| 09 | Directors and Film Pipeline | Director's notes, revision etiquette, animating to editorial, Anim Sup |
| 10 | Demo Reel | Shot selection, ordering, music, length, studio submission requirements |

## Prerequisites

- Basic knowledge of a 3D animation package (Maya, Blender, or equivalent)
- Completion of an introduction to animation (12 principles)
- Familiarity with the graph editor concept
- A drive to work at the highest professional level

---

## The Pixar Standard

Every piece of content in this course is held to the standard articulated by Pixar's "Notes on a Scene" video series: can you articulate *why* you made every performance choice? If the answer is "it felt right," the work isn't done yet.
