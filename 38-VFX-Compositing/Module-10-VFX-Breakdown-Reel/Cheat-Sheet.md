# 🗒️ Module 10 Cheat Sheet: VFX Breakdown Reel & Career

## Reel Standards

| Attribute | Standard |
|-----------|---------|
| Length | 60–120 seconds |
| Shot count | 4–8 shots |
| First shot | Your absolute best |
| Intro | None — cut directly to work |
| Format | H.264/H.265 MP4 or ProRes 422 |
| Resolution | 1920×1080 minimum |
| Breakdown | Before/after for every shot |

---

## Reel Rejection Reasons

| Problem | Fix |
|---------|-----|
| No breakdown (comp only) | Show plate + elements + comp |
| Floating CG (no shadow/light integration) | Fix interactive light + shadow catcher |
| Mismatched grain | Single grain Adjustment Layer on top |
| Color-cast elements | Grade each element to match plate |
| AE only (for film positions) | Add at least one Nuke shot |
| Team shot with unclear ownership | Label exactly what you did |

---

## Career Tracks

| Track | Focus | Primary Tools | Employers |
|-------|-------|--------------|-----------|
| Compositor | Shot finishing, integration | Nuke, AE | ILM, Weta, Framestore, boutiques |
| VFX Artist (FX) | Element creation, simulation | Houdini, Blender, Particular | Same + FX-specialist studios |
| Technical Director | Pipeline, scripting | Python, Nuke TCL, C++ | All major studios |

## Reel Shot Variety Targets

1. Greenscreen key (clean edge, spill, light wrap)
2. Roto/matte (complex subject, Nuke preferred)
3. 3D CG integration (tracked, correctly lit)
4. Particle FX (fire, smoke, or magic)
5. Invisible effect (set ext, sky, cleanup)
6. Action/destruction FX (layered explosion or muzzle flash)
