# 🗒️ Module 10 Cheat Sheet: VFX Breakdown Reel & Career

## Reel Technical Standards

| Attribute | Standard |
|-----------|---------|
| Total length | 60–120 seconds maximum |
| Shot count | 4–8 shots |
| First shot | Absolute best work |
| Intro | None — cut directly to work |
| Video codec | H.264/H.265 MP4 or ProRes 422 |
| Resolution | 1920×1080 minimum; 3840×2160 preferred |
| Frame rate | 24fps (film) or match source |
| Audio | Licensed or original music only |
| Watermark | Name + contact; subtle; corner |
| File size | Under 500MB for online submission |
| Breakdown | Before/after for EVERY shot |

---

## Reel Breakdown Format Options

| Format | Duration | Best For |
|--------|---------|---------|
| Split-screen (plate left, comp right) | 5–8 sec | Keying, sky replacement |
| Wipe left-to-right | 5–8 sec | Before/after across frame |
| Side-by-side: plate / elements / final | 8–12 sec | Complex multi-element comp |
| Breakdown sequence | 10–20 sec | Explosion, destruction sequence |

**Correct order: plate → elements → final. Never show final first.**

---

## Reel Rejection Reasons

| Problem | Fix |
|---------|-----|
| No breakdown (comp only shown) | Show plate + elements + comp for every shot |
| Floating CG (no shadow/light) | Fix interactive light + shadow catcher |
| Mismatched grain | Apply single grain Adjustment Layer at top of stack |
| Color-cast elements | Grade each element to match plate |
| AE only (for film positions) | Add at least one Nuke shot |
| Team shot with unclear ownership | Label exactly what you did (e.g., "compositing: roto, keying") |
| Reel over 2 minutes | Cut to your 6–8 best shots; delete the rest |
| Slow intro titles or music fade | Cut instantly to work |

---

## Shot Variety Targets (4+ categories required)

| Skill Category | What to Show |
|---------------|-------------|
| Greenscreen key | Clean edge, spill suppression, light wrap |
| Roto/matte | Complex subject; Nuke preferred for film positions |
| 3D CG integration | Tracked, correctly lit, correct shadows |
| Particle FX | Fire, smoke, or magic — demonstrate Particular knowledge |
| Invisible effect | Set extension, sky replacement, cleanup |
| Destruction/action FX | Layered explosion or muzzle flash |

| Studio | Primary Work | Must-Have Skills | Nice-to-Have |
|--------|-------------|-----------------|--------------|

## Career Tracks Comparison

| Track | Focus | Primary Tools | Progression |
|-------|-------|--------------|-------------|
| Compositor | Shot finishing, integration | Nuke, AE | Junior → Mid → Senior → Lead → VFX Supervisor |
| VFX Artist (FX) | Element creation, simulation | Houdini, Blender, Particular | Junior → FX Artist → Senior → FX Supervisor |
| Technical Director | Pipeline, scripting | Python, Nuke TCL | TD → Pipeline Supervisor |

## Major Studio vs Boutique

| Factor | ILM / Weta / Framestore | Boutique |
|--------|------------------------|---------|
| Specialization | Deep | Generalist |
| Shot complexity | Hero/Tier 4 | Tier 2–3 |
| Job security | Higher | Project-dependent |
| Advancement | Slower | Faster (3–5 yrs to Lead) |
| Salary | Higher | Lower |
| Learning | Extraordinary depth | Rapid generalist |

## ILM Endgame Breakdown Format (Industry Standard Reference)

| Segment | Duration | Content |
|---------|---------|---------|
| Hero wide shot | 15 sec | Plate → tracking markers → portal elements → CG armies → final |
| Character close-up | 10 sec | Performer plate → wireframe → skin render → comp |
| Environmental FX | 8 sec | Environment → particles → destruction → grade |

**Show in order: raw → elements → final. This is the accepted professional format.**
