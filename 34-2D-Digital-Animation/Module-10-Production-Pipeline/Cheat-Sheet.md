# ⚡ Module 10 Cheat Sheet, Production Pipeline

---

## 2D Studio Pipeline Stages

| # | Stage | Key Output |
|---|-------|-----------|
| 1 | Development | Concept, bible, character designs, color script |
| 2 | Pre-Production | Scripts, storyboards, locked animatic |
| 3 | Layout | Camera staging, background placement, character guides |
| 4 | Rough Animation | Key poses, rough in-betweens (timing & performance) |
| 5 | Cleanup | Clean, on-model lines redrawn over roughs |
| 6 | Ink & Paint / Color | Digital color fill applied to clean lines |
| 7 | Composite | Characters + BG + effects assembled; color grade |
| 8 | Sound & Music | Final audio mix |
| 9 | QC / Delivery | Quality check → master delivery |

---

## Animatic: The Most Important Stage

```
Change cost multiplier:
  Storyboard:     1×
  Animatic:       3×
  Rough animation: 10×
  Cleanup/color:  30×
  Composite:      50×
  Delivery:       100× (sometimes impossible)
```

The animatic catches timing, pacing, and story problems **before any animation is drawn.**

## Harmony vs. Animate

| | Harmony | Animate |
|-|---------|---------|
| Price | Higher ($109–$239/mo) | Lower (CC) |
| Pipeline | Complete (draw → composite) | Needs AE for composite |
| Industry use | Major broadcast studios | Commercial, indie, YouTube |
| Deformers | Advanced | Standard |
| Learning curve | Steeper | More accessible |
| File format | .xstage | .fla / .an |
| Collaboration | Database system | File-based |

## Model Sheet Contents

- Front, 3/4, side, back views
- Multiple expressions and emotional states
- Size comparison chart (relative to other characters)
- Exact color specifications (hex codes or Pantone)
- Notes on stylistic rules (hair direction, proportions)

---

## Color Script

A visual map assigning a palette to each scene for emotional consistency across the full production.

| Color Script Role | Function |
|-----------------|---------|
| Scene-by-scene swatches | Quick overview of entire episode's color journey |
| Warm/cool temperature | Warm = safety/home; cool = threat/isolation |
| Character palette consistency | Heroes vs. villains visually distinct |
| Special event callouts | Action, flashbacks, emotional peaks |
