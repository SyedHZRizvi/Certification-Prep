# 🏆 Final Mock Exam, Adobe Certified Professional: Animate
## 2D Digital Animation · 30 Questions · 50 Minutes

**Instructions:** This written mock mirrors the length and pacing of the real **Adobe Certified Professional in Animate** exam, which runs **50 minutes** with **~30 questions** and requires a scaled score of **700/1000**. Aim for 85%+ here before booking. Each question has one correct answer.

> 🚨 **Read this before you start.** The real exam has **two sections**: a **Questions** section (selected-response, which this mock rehearses) *and* a **Tasks** section of **live in-application activities you perform inside Adobe Animate** — building, modifying, and publishing in the actual software, scored on what you produce. **A written mock cannot replicate the live Tasks section.** Treat 85%+ here as evidence you have the *concepts*; you are only fully ready once you can also perform the module labs (create symbols, set audio sync, build tweens, rig with the bone tool, publish) from memory inside Animate. This mock is honestly half your preparation, not all of it.

> 📌 The After-Effects material in Modules 5–6 is supplementary cross-tool craft and is **not** on the ACP Animate exam, so it is **not tested below**. Every question maps to one of the five official Animate objective domains.

## 🎯 Answer Key (No Cheating!)

```
1.  C   Guide layers are visible to the animator but never appear in exports.
2.  C   Graphic symbols sync to the parent timeline, correct for walk cycle + dialogue sync.
3.  C   F8 converts selected artwork to a symbol.
4.  B   Stream sync ties audio to the timeline playhead, required for lip sync/scrubbing.
5.  C   In Merge Drawing Mode, different-colored overlapping shapes cut into each other.
6.  B   Event sync plays to completion regardless of timeline position, correct for BG music.
7.  B   Wrong registration point at bounding-box center causes rotation around the wrong pivot.
8.  B   Enter/Return plays and stops animation in Adobe Animate.
9.  B   On twos = new drawing every 2 frames = 12 unique drawings/second at 24fps.
10. C   Cleanup = clean, on-model lines drawn over rough animation drawings.
11. C   Well-executed smear frames feel invisible during playback.
12. A   Crowded end, spread start = motion still accelerating (ease in / slow start).
13. B   Pose-to-pose: key poses first, then in-betweens fill the spaces.
14. B   Classic Tween ease -100 = ease in (slow start, fast finish).
15. C   Classic Tweens use Motion Guide layers for drawn path following.
16. B   Orient to Path auto-rotates the symbol to face the direction of travel.
17. C   IK: dragging the foot (end of chain) poses the limb; the chain follows.
18. B   Shape tweens morph raw drawn shapes (not symbols) on the same layer.
19. C   AI = wide open mouth with visible teeth (sounds like "say," "eye").
20. C   CDG = slightly open with active tongue position for D, G, S, T, Z.
21. A   "People": P (MBP) → ee (E) → p (MBP) → l (L).
22. B   REST = silence, pause, exhale.
23. B   Contact → Down → Passing → Up (correct four-pose order).
24. B   12-frame cycle: Contact at 1, Down at 3.
25. C   Anticipation for an upward jump = crouch downward first (opposite direction).
26. A   Follow-through = secondary elements continuing after primary motion stops.
27. B   Animate publishes to HTML5 Canvas, a web-native scalable target.
28. B   On twos (12 drawings/sec) gives the characteristic hand-drawn/anime look.
29. C   The animatic cuts storyboard to audio and catches timing problems before animation.
30. B   Model sheets ensure character consistency across all animators.
```
