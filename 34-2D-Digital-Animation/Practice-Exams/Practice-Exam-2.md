## 🎯 Answer Key (No Cheating!)

```
1.  B   DUIK is a free, open-source AE rigging plugin from RxLaboratory.
2.  C   The foot goal (end effector) controller is what the animator keyframes; IK solves the rest.
3.  B   Spring rig = procedural secondary animation (lag and oscillation) without keyframes.
4.  B   Stiffness controls how quickly the spring returns to rest after displacement.
5.  C   Standard J+S = 5 positions: center, up, down, left, right.
6.  B   Rubber hose = continuously curved limbs with no defined joint break.
7.  B   The animator draws all five face positions; J+S blends between them.
8.  C   FV = upper teeth on lower lip (F and V sounds).
9.  C   Stream sync ties audio to playhead, required for lip sync.
10. B   Character Animator requires layered .psd or .ai files with named layers.
11. B   REST = silence, pause, exhale, neutral resting position.
12. B   Four walk poses: Contact, Down, Passing, Up.
13. C   12-frame cycle: second Contact at frame 9.
14. C   Cuphead mimics Fleischer Brothers (1930s Betty Boop, Popeye aesthetic).
15. B   Waterbending = Tai Chi Chuan (flowing, circular movements).
16. C   Anticipation always moves opposite to the main action direction.
17. C   AT impact = maximum squash (maximum compression at impact).
18. B   H.264 requires Adobe Media Encoder, not AE Render Queue directly.
19. C   Lottie requires Shape Layers (vector path data); raster footage won't export.
20. B   ProRes 4444 = alpha channel + high quality for compositing pipelines.
21. B   sRGB and Rec.709 share primaries; their gamma encoding curves differ.
22. D   TikTok = 9:16 portrait (1080×1920).
23. B   Animatic = storyboard cut to audio; identifies timing/pacing before animation.
24. C   Harmony is used at major broadcast studios (Simpsons, Rick and Morty).
25. B   Layout establishes camera staging, shot composition, and character placement.
26. B   TVPaint = high-quality hand-drawn; used by Cartoon Saloon and similar studios.
27. B   Color script = palette-to-scene map for emotional storytelling consistency.
28. B   Cleanup = clean, on-model lines drawn over roughs for animator delivery.
29. B   GIF's 256-color limit causes banding; WebM supports full color + alpha.
30. B   Manual override = keyboard-triggered mouth shapes to correct auto-sync errors.
```
