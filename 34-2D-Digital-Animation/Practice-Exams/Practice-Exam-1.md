## 🎯 Answer Key (No Cheating!)

```
1.  B   Objects entirely outside the Stage don't appear in exports.
2.  C   Movie Clips have independent timelines; Graphic symbols sync to parent.
3.  C   On twos at 24fps = 12 unique drawings per second.
4.  C   Stream sync ties audio to the playhead — required for lip sync.
5.  B   F7 inserts a blank keyframe (hollow circle).
6.  C   Registration point at the joint (shoulder) for correct rotation.
7.  B   Crowded start, spread end = decelerating motion (ease out).
8.  B   Smear frames simulate fast motion blur.
9.  C   The Motion Editor is exclusive to Motion Tweens.
10. B   Floaty = linear easing, no holds, no overshoot.
11. C   Motion Tween paths are bezier curves.
12. B   "Back Out" overshoots the destination then settles back.
13. B   +100 ease value = strong ease out (fast start, slow finish).
14. B   FK = rotate each joint parent-to-child independently.
15. C   MBP = closed or nearly closed (M, B, P bilabial sounds).
16. B   Swap Symbol preserves position, scale, and rotation.
17. C   Hazbin Hotel uses Animate/Flash-style cut-out rigs.
18. B   "Create Shapes from Vector Layers" converts AI layers to native AE Shape Layers.
19. B   Null Objects are invisible at render time; used for rig control.
20. B   The Pick Whip creates parent-child layer relationships.
21. C   F9 applies Easy Ease to selected keyframes.
22. B   Speed Graph shows velocity (rate of change).
23. B   Import as Composition — Retain Layer Sizes keeps individual layers.
24. B   Puppet Pin Tool is for mesh deformation on a single flattened layer.
25. A   Pre-composing = packaging layers into a reusable nested comp = symbol equivalent.
26. B   Cleanup layer goes above roughs; clean lines trace over the rough guide.
27. C   Per-property curves in the Motion Editor give each attribute its own timing.
28. B   Contact pose = heel of lead foot touching the ground.
29. B   Guide layers are visible to the animator but never appear in exports.
30. B   Swap Symbol preserves the instance's position, scale, and rotation.
```
