## 🎯 Answer Key (No Cheating!)

```
Q1:  B — Trim Paths (under Contents > Shape in a shape layer) is the native AE tool for the draw-on effect. It animates the visible portion of a stroke path.

Q2:  B — Path morphs require matching point counts. AE maps Point 1 to Point 1, etc. — mismatched counts cause tearing or twisting.

Q3:  B — Y-axis rotation takes the logo from edge-on (90°) to full face (0°), simulating a flip that reveals the face. X-axis would be a top-to-bottom flip.

Q4:  C — The Netflix N ident uses separate 3D layers representing strips of the N that separate in Z-space (depth) before collapsing to the flat 2D form.

Q5:  D — Instagram Story templates are social media assets, not broadcast package components. Opens, bumpers, and lower-thirds are broadcast package staples.

Q6:  C — Ease-In: the Swoosh accelerates as if gaining brush momentum, reaching full speed at the tip of the stroke. This mimics athletic energy.

Q7:  B — Title-safe zone = 80% of frame dimensions. Action-safe = 93.75%. Text and logos should stay within the 80% title-safe region.

Q8:  B — Font stacks for web usage are a web design decision, not part of a brand motion vocabulary document. Motion docs cover easing, speed, direction, color-in-motion, sound, and prohibited motions.

Q9:  B — Layer > Create > Create Shapes from Vector Layer converts Illustrator artwork into editable AE shape layers, enabling path-level animation.

Q10: B — Trim Paths End at 0% = no visible path. End at 100% = full path visible. Animating from 0% to 100% creates the draw-on effect.

Q11: C — Blue, Red, Yellow, Green is Google's canonical color order, maintained consistently across all animated logo variants.

Q12: C — Broadcast legal luminance: 16 (black) to 235 (white) on a 0–255 scale. Values outside this range can cause signal issues on broadcast transmission.

Q13: B — Shape morphing is the natural technique for geometric transformations between abstract forms. Draw-on requires a clear stroke path; 3D flip requires a flat mark.

Q14: B — AE's default morph maps corresponding-numbered points. If Point 1 on Shape A is at the top and Point 1 on Shape B is at the bottom, the path twists during the morph.

Q15: B — Most brand guidelines specify minimum opacity levels for logo visibility. Dipping to 5% would violate the "always visible/legible" rule in almost every professional brand standard.

Q16: D — Rec. 709 (BT.709) is the standard color space for HD broadcast. Using sRGB can result in color and luminance discrepancies when the content reaches broadcast screens.

Q17: B — A bumper is a short (3–5 second) transition stinger used going into or returning from commercial breaks. It's a package element, not a full open.

Q18: B — The Netflix "tudum" sound was introduced in 2019 and is precisely synchronized to the moment the N layers snap to their final flat position — creating an audio-visual payoff.

Q19: B — Apply separate Trim Paths to each shape and offset their keyframe timing independently. Each shape has its own animation, allowing independent start times.

Q20: C — Flipping logos upside down or mirroring them is almost universally prohibited in brand guidelines. Transparency fade-outs, draw-ons, and dark backgrounds are all standard approved uses.

Q21: B — Action-safe = 93.75% of frame. Title-safe = 80%. Moving subjects can approach action-safe; text and logos must stay inside title-safe.

Q22: A — Duik Ángela's main advantage for morphing is that it can interpolate between shapes with different point counts, which native AE path morphing cannot do cleanly.

Q23: C — Timing. How fast or slow a brand moves is directly a timing decision. The "speed signature" maps to animation duration and frame counts — the heart of Principle 9.

Q24: C — MOV with alpha channel (ProRes 4444 or PNG codec) preserves transparency, allowing the logo to be composited over any background in a broadcast/post context.
```
