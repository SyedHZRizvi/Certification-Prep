# ⚡ Module 9 Cheat Sheet — Sound Design & Motion

## Three Layers of Sound

| Layer | Volume (relative) | Purpose |
|-------|------------------|---------|
| Music | -6 to -12dB (under VO) | Emotional foundation |
| SFX | -3 to 0dB | Event-specific punctuation |
| Ambient | -12 to -18dB | Textural background |
| Voice-Over | 0dB (reference) | Information delivery |

---

## Sync Rules

| Rule | Detail |
|------|--------|
| Hit Point | Visual event coincides with audio transient |
| Tolerance | ±2 frames = perceived as perfect sync |
| Lead/Lag | Lead audio (visual early) = tight feel; lag audio = sluggish |
| Beat Markers | Use Ctrl+click on timeline at each beat |
| Convert Audio to Keyframes | Layer > Audio > Convert Audio to Keyframes |

## Audio Spectrum Setup (AE)

1. Import audio → place in comp
2. Create solid layer
3. Effect > Generate > Audio Spectrum
4. Audio Layer: [your audio]
5. Frequency Bands: 32–128 (more = more detail)
6. For circular: create ellipse mask on solid first

### Key Parameters

| Parameter | Controls |
|-----------|---------|
| Maximum Height | Bar height in pixels |
| Softness | Bar top blur (0 = sharp) |
| Frequency Bands | Number of bars |
| Dynamic Hue Phase | Animates color with amplitude |

---

## Tone of Motion Matrix

| Audio | Animation Style |
|-------|----------------|
| High-energy electronic | Fast, sharp, linear, high contrast |
| Orchestral/dramatic | Slow, arcing paths, high exaggeration |
| Ambient/minimal | Slow, smooth, subtle wiggle |
| Corporate/professional | Medium, ease-in-out, structured |
| Playful/quirky | Variable speed, spring easing, color |

---

## Sound Design Workflow

```
1. Choose music (sets emotional tone)
2. Mark beats (Ctrl+click markers on timeline)
3. Build animation to beat structure
4. Add SFX at key visual moments
5. Mix (music -6 to -12dB, SFX -3 to 0dB, VO at 0dB)
6. "Sweeten" with subtle ambient SFX
```
