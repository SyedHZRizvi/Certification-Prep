# Module 4 Cheat Sheet — روزمرہ گفتگو (Everyday Conversations)

Phrasebook organized by situation for quick reference.

---

## Shopping / Bazaar (بازار)

| Urdu | Transliteration | English |
|------|-----------------|---------|
| کیا دکھاؤں آپ کو؟ | Kya dikhaoon aap ko? | What shall I show you? |
| یہ دکھائیں | Yeh dikhaain | Show me this |
| کتنے کا ہے یہ؟ | Kitne ka hai yeh? | How much is this? |
| بہت مہنگا ہے | Bahut mehnga hai | It's too expensive |
| کچھ کم کریں | Kuch kam karein | Reduce it a bit |
| آخری قیمت کیا ہے؟ | Aakhri qeemat kya hai? | What is the final price? |
| میں لے جاتا/جاتی ہوں | Main le jaata/jaati hun | I'll take it |
| منظور | Manzoor | Agreed / Deal |
| پیک کر دیں | Pack kar dein | Please pack it |
| بازار قیمت | Baazaar qeemat | Market price |
