# 📋 Module 4 Cheat Sheet: Meta Ads Mastery

> One page. Print it. Tape it to your monitor. Review before every Meta launch.

---

## 🏛️ Hierarchy

```
Business Manager (admin hub)
└── Ad Account
    └── Campaign  ← OBJECTIVE, BUDGET (if CBO)
        └── Ad Set  ← AUDIENCE, PLACEMENTS, BUDGET (if ABO)
            └── Ad  ← CREATIVE, COPY, CTA
```

🧠 **Never** set up via personal Facebook profile. Business Manager always.

---

## 📊 Day-Zero Setup (Required)

```
☐ Business Manager created
☐ Domain Verified
☐ Pixel installed (GTM)
☐ CAPI configured (server-side)
☐ Event Deduplication enabled
☐ AEM events ranked (top 8)
☐ Customer list uploaded
☐ Catalog (if ecom)
```

---

## 🔧 Pixel + CAPI

| | Pixel | CAPI |
|---|-------|------|
| Where | Browser | Server |
| Blocked by ATT | ✅ Yes | ❌ No |
| Coverage | 40-60% | 85-95% |
| Setup | Easy | Medium |

**Use BOTH with deduplication.**

---

## 👥 Audiences (3 Tiers)

```
CUSTOM AUDIENCE  →  Built from YOUR data (cust list, visitors, video viewers)
LOOKALIKE        →  Meta-generated similar (1% closest, 3% scale, 10% reach)
ADVANTAGE+       →  Meta picks; your hint is optional
```

---

## 🛍️ Advantage+ Shopping (ASC)

| Use ASC | Use Manual |
|---------|------------|
| 50+ daily purchases | < 50 daily purchases |
| Have catalog | No catalog |
| $50+/day budget | < $20/day |

**Existing Customer Budget Cap** → set 30% to force prospecting.

---

## 🧪 DCT Framework

```
HOOK options × BODY options × CTA options
Cap: 4 × 4 × 4 = 64 combos max
Min impressions before judging: ~1,000/variant
```

Test for 7+ days at $50+/day for real signal.

## 📐 Campaign Structure for Testing

```
Campaign 1: TEST (ABO ad-set-level budget)
  4 ad sets × different audiences × SAME 4 ads
Campaign 2: SCALE (CBO, campaign-level budget)
  Winners from Test
Campaign 3: RETARGETING ($3-5/day always on)
  Cart abandoners + ATC + video viewers
```

## ⚠️ Top Mistakes

- ❌ Personal FB ad account
- ❌ No CAPI = broken tracking
- ❌ Editing in learning phase (under 50 conv)
- ❌ 20+ tiny ad sets ($5/day each)
- ❌ Killing creative at 100 impressions
- ❌ Single-placement (use Advantage+ Placements)
- ❌ No retargeting campaign
- ❌ Cropping Feed creative to Reels
