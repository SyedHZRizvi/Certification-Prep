## 🔬 The 4-Stage CRO Cycle

```
1. OBSERVE     (Hotjar / Clarity replays + heatmaps)
2. HYPOTHESIZE (one specific change, one expected outcome)
3. TEST        (A/B, proper sample size)
4. ITERATE     (roll out, then next problem)
```

🚨 Skipping step 1 = guessing.

---

## 📊 Bayesian vs Frequentist

| | Frequentist | Bayesian |
|---|------------|----------|
| Output | p-value | "Probability best" % |
| Sample size | Fixed upfront | Flexible |
| Peeking | ❌ inflates errors | ✅ OK |
| Common in | Optimizely Web | VWO Bayesian, Convert |

### P-value clarification

`p < 0.05` means: **probability of seeing this data IF no real effect < 5%**.
NOT: "95% chance treatment is better."

---

## 🧮 Sample Size Inputs

```
1. Baseline conversion rate
2. MDE (Minimum Detectable Effect, relative)
3. Statistical power (0.80)
4. Alpha (0.05)
```

Calculator: **evanmiller.org/ab-testing/sample-size.html**

### Reference table

| Baseline | 10% MDE | 20% MDE |
|----------|---------|---------|
| 1% | ~155K/var | ~38K/var |
| 2% | ~77K/var | ~19K/var |
| 5% | ~30K/var | ~7.5K/var |
| 10% | ~14K/var | ~3.5K/var |

## 🎨 Landing Page Anatomy (8 Sections)

```
1. HERO       , Headline + subhead + ONE CTA
2. SOCIAL PROOF, Logos OR star rating
3. PROBLEM    , Pain agitation
4. SOLUTION   , 3 features → 3 benefits
5. PROOF      , Real testimonial + photo
6. OBJECTIONS , FAQ (3-4)
7. PRICING    , Anchor high, recommend middle
8. FINAL CTA  , Repeat + risk reversal
```

## ⚠️ Top CRO Mistakes

- ❌ Test without observation first
- ❌ Stop test before sample size
- ❌ Change variant mid-run
- ❌ Test low-traffic pages
- ❌ Ignore mobile
- ❌ Multiple variables per test
- ❌ Send paid traffic to homepage
- ❌ Stock handshake photos
