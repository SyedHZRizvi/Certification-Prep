# Module 10 Cheat Sheet: Portfolio & Career 📋

---

## Demo Reel Rules

| Rule | Detail |
|------|--------|
| **Open with best shot** | Recruiters decide in first 10 seconds |
| **90 seconds ideal** | 2 minutes absolute maximum |
| **Every shot = strong** | Weakest shot sets your perceived level |
| **Tailor per role** | Character reel for character role; creature for creature |
| **Label everything** | Shot name, software, your contribution if collaborative |
| **No filler** | Never include work you're not proud of |
| **No turntables** | Unless you're applying as a rigger/modeler |

## Reel Shot Types (What to Include)

| Shot Type | Why |
|-----------|-----|
| Character pantomime | Proves acting ability — highest value |
| Walk cycle with personality | Industry standard test |
| Dynamic action | Weight and momentum |
| Dialogue/lip sync | Required for character animation roles |
| Creature locomotion | High-value differentiator |
| Facial performance | Subtle emotion |

## Platform Selection

| Platform | Best For | Notes |
|----------|---------|-------|
| Vimeo | Reel hosting | Industry preferred; ad-free; password protection |
| ArtStation | Game/film portfolio | Dominant in game art industry |
| Behance | Motion design | Less for character animation |
| LinkedIn | Professional network | Link to Vimeo; not primary host |
| YouTube | Public tutorials | Less professional for primary reel |

**Standard setup:** Vimeo (reel) + ArtStation or personal site (full portfolio) + LinkedIn (network)

## Freelance vs Studio Quick Compare

| | Studio | Freelance |
|-|--------|-----------|
| Stability | High | Low (feast/famine) |
| Benefits | Yes (employer-provided) | No (self-provided) |
| Creative control | Low | Higher |
| Income ceiling | Salary bands | Unlimited (in theory) |
| Tax complexity | Simple | Complex (quarterly) |

**Pattern:** Studio first → Freelance later (once established)

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| Longer reel shows more range | Weakest shot in reel = your perceived level; cut ruthlessly |
| YouTube = fine for reels | Vimeo is industry preferred; YouTube is less professional |
| Turntables are good reel content | Turntables show modeling; not animation performance |
| Studio = creative freedom | Studio = executing the director's vision; freelance = more creative control |
| All jobs are publicly posted | 75% of jobs are filled through the network; networking is essential |
