# Module 5 Cheat Sheet: Timing, Spacing & Rhythm 📋

---

## Frame Rates at a Glance

| Rate | Context | Notes |
|------|---------|-------|
| 24 fps | Film (standard) | "Cinematic look" |
| 25 fps | PAL TV (Europe) | Slightly faster than film |
| 30 fps | NTSC TV (North America) | Broadcast standard |
| 12 fps (on 2s) | Traditional 2D animation economy | Each drawing shown twice |
| 60 fps | Gaming | Real-time standard |

**On 1s:** Every frame is a unique drawing (24 drawings/sec at 24fps)
**On 2s:** Every drawing shown for 2 frames (12 drawings/sec at 24fps)

---

## Timing Ladder (Emotional Reference)

| Duration | Frames @24fps | Feel |
|----------|--------------|------|
| 2–4 frames | Lightning | Shock, violence, comedy snap |
| 6–8 frames | Quick | Surprise, energy |
| 10–16 frames | Normal | Standard life-size action |
| 20–32 frames | Deliberate | Controlled, weighted |
| 36–48 frames | Heavy | Fatigue, grief beginning |
| 60–96 frames | Ceremonial | Grand weight, ritual |
| 120+ frames | Unbearable | Grief, isolation |

---

## Timing Chart Spacing Key

```
SLOW-IN (from A):    A 1 2 — 3 ——————— B  (clustered near A)
SLOW-OUT (to B):     A ——————— 1 — 2 3 B  (clustered near B)
SLOW IN + OUT:       A 1 ——— 2 ——— 3 B  (S-curve distribution)
CONSTANT (linear):   A — 1 — 2 — 3 — B  (even spacing)
```

## Graph Editor F-Curve Shapes

| Curve Shape | What It Produces |
|------------|-----------------|
| Straight diagonal | Constant speed (mechanical) |
| S-curve | Slow-in/slow-out (ease in/ease out) |
| Flat then steep | Slow-in (eases out of start) |
| Steep then flat | Slow-out (eases into end) |
| Horizontal flat | Hold (no movement) |

**The Graph Editor Trap:** Beautiful, smooth S-curves on every parameter = dead, statue-on-turntable motion. Real animation is messier.

---

## Rhythm Rules

- **Rule of three:** 3 repetitions create expectation → 4th variation creates surprise
- **Hold + action:** Holds before a big action = build anticipation; holds after = register the moment
- **Fast-slow contrast:** Rhythm is the pattern; contrast within the pattern creates energy

---

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| Slow timing = drama, fast = comedy | Same action at different timings can be either; depends on scene goal |
| Default S-curve = good timing | S-curve may look right but feel dead; adjust for your specific performance |
| "On 2s" = only 2 frames total | On 2s = every drawing shown for 2 frames (12 drawings/sec at 24fps) |
| PAL = 24fps | PAL = 25fps (Europe); NTSC = 30fps (North America) |
| Spacing = timing | Spacing = how far drawings move in space; timing = how many frames between them |
