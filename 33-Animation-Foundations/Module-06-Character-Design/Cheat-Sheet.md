# Module 6 Cheat Sheet: Character Design 📋

---

## Shape Language Reference

| Shape | Association | Best For |
|-------|------------|---------|
| Circle / Sphere | Friendly, approachable, safe, youthful | Heroes, children, sidekicks, comic characters |
| Square / Rectangle | Stable, reliable, powerful, stubborn | Mentors, father figures, authority |
| Triangle / Point | Dangerous, menacing, dynamic, unstable | Villains, tricksters, fast characters |

**Examples:**
- Mickey Mouse = circles everywhere → universally approachable
- Scar (Lion King) = elongated triangles → threatening
- Mufasa (Lion King) = rectangular shoulders → reliable authority

---

## Silhouette Test

```
STEP 1: Fill character design with solid black
STEP 2: Ask, can you identify the character?
STEP 3: Ask, can you read what they are doing?

PASS: Design is strong; readable at distance and small scale
FAIL: Design needs simplification or more distinctive poses
```

## Color-Personality Reference

| Color | Association | Example |
|-------|------------|---------|
| Red | Energy, passion, danger | Moana's dress (energy) |
| Blue | Calm, trustworthy, melancholy | Elsa (isolation/cold) |
| Yellow/Gold | Warmth, wisdom, optimism | Belle's dress, Mufasa's mane |
| Green | Nature, envy, unnatural | Maleficent (supernatural) |
| Purple | Power, mysticism, villainy | Ursula, Thanos |
| Orange | Playful, adventurous | Nemo, Tigger |
| Black/Dark | Mystery, villainy, sophistication | Scar, Maleficent |

**Note:** These are conventions, not rules. *Arcane*'s Jinx uses blues/purples as a protagonist, deliberate subversion.

---

## Character Turnaround Views

1. Front (0°)
2. 3/4 view (45°)
3. Side / Profile (90°)
4. 3/4 Back (135°)
5. Back (180°)

All views must maintain consistent: proportions, volume, joint locations

---

## Designing for Movement Checklist

- [ ] Joints clearly visible in design (know where limbs bend)
- [ ] Detail concentrated on non-deforming areas (face, hands)
- [ ] Hair does not obscure face in extreme poses
- [ ] Silhouette reads in extreme action poses
- [ ] Slight asymmetry (not perfectly symmetrical)
- [ ] Passes silhouette test in 3 typical poses

---

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| Purple = royalty only | Purple also = villainy, power, mysticism |
| Silhouette test = check black-and-white color | Silhouette test = fill with solid BLACK and check readability |
| IP = international proportions | IP = Intellectual Property (copyright/trademark) |
| Circle = hero, triangle = villain always | These are tools; Arcane's Jinx shows they can be subverted |
| Freelancer always owns their designs | Ownership depends on contract, always clarify in writing |
| Turnaround = action pose sheet | Turnaround = rotation views (front, 3/4, side, back, 3/4 back) |
