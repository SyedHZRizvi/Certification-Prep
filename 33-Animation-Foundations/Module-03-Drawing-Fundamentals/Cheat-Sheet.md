# Module 3 Cheat Sheet: Drawing Fundamentals 📋

---

## Gesture Drawing Hierarchy

```
1. LINE OF ACTION, the first mark; the dominant energy direction
2. MAJOR VOLUMES, head (egg), torso (box), pelvis (wedge)
3. JOINTS, shoulder, elbow, wrist, hip, knee, ankle
4. LIMB CYLINDERS, connect joints; modified cylinders
5. REFINEMENT, only after structure is established
```

---

## Proportion Systems

| Style | Head:Body | Used in |
|-------|-----------|---------|
| Realistic | 1:7.5 | Film VFX characters |
| Heroic | 1:6 | Disney heroes, action games |
| Standard Animation | 1:5 | Most animated series |
| Infant/Toddler | 1:2.5 | Child characters |
| Chibi / Super-Deformed | 1:1 to 1:2 | Anime chibis, *Gravity Falls* |

---

## Foreshortening Rules

- Limb pointing **toward** camera = appears **shorter** than actual
- Object **closer** to camera = appears **larger**
- Object **farther** from camera = appears **smaller**
- Use **overlap** to establish depth: closer objects overlap farther ones
- **The Fist Test:** A punch toward camera, fist should be LARGER than the head at full extension

---

## Life Drawing: What It Builds

| Skill | How |
|-------|-----|
| Observational accuracy | You see what's actually there |
| Speed & decision-making | Timed sessions force prioritization |
| Weight & balance | Real figures must balance |
| Hand-eye coordination | Mark-making becomes automatic |

**Recommended frequency:** 20–30 minutes daily, 3–5x/week.

---

## Common Drawing Mistakes

| Mistake | Fix |
|---------|-----|
| Drawing from wrist | Draw from shoulder |
| Outline before structure | Construction first, outline never required |
| Symmetrical poses | Shift weight; vary arm heights |
| Tangents (two shapes touching at one point) | Use overlap instead |
| Drawing parts separately | Line of action first, parts serve the whole |
| Copying reference literally | Study-then-draw: observe, close, draw from understanding |

---

## Reference Use (Professional Method)

```
WRONG: Open reference → copy frame by frame → animation looks rotoscoped

RIGHT:
1. Study reference for 5–10 minutes (understand the physics)
2. Close the reference
3. Thumbnail the movement from understanding
4. Return to reference only for specific detail verification
```

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| "Using reference is cheating" | Every professional uses reference; HOW you use it matters |
| Gesture = detailed life drawing | Gesture = capturing energy quickly, not anatomy accurately |
| Symmetrical = stable = good | Symmetrical = lifeless in animation; vary weight distribution |
| Construction = cartoon style only | Construction applies to all styles, including realistic |
| Chibi proportions = 5:1 | Chibi = 1:1 to 2:1; 5:1 is standard animation proportion |
