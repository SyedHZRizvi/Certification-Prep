# Module 7 Cheat Sheet: Color Theory & Composition 📋

---

## Color Properties

| Property | Definition | Animation Impact |
|----------|-----------|-----------------|
| **Hue** | The color name (red, blue, green) | Character identity, emotional association |
| **Value** | Lightness/darkness | Readability, contrast, depth — MOST IMPORTANT |
| **Saturation** | Intensity/purity | Emotional volume: high = energy, low = fatigue |

**Pixar Rule:** Value first. Compositions that work in grayscale work in color.

---

## Color Harmony Systems

| System | Description | Feel |
|--------|-------------|------|
| Monochromatic | One hue, multiple values | Unity, restriction |
| Analogous | Adjacent wheel colors | Natural, comfortable |
| Complementary | Opposite wheel colors | Tension, drama, vibrance |
| Split Complementary | One color + two flanking its complement | Contrast with less tension |
| Triadic | Three equally spaced (120°) | Dynamic, complex |

**Most common pro choice:** Analogous palette + complementary accent

---

## Value Palettes for Mood

| Palette | Mood | Example |
|---------|------|---------|
| High-key (light) | Joy, innocence, safety | Tangled tower sequences |
| Low-key (dark) | Danger, menace, grief | Coco underworld; Maleficent scenes |
| High-contrast | Drama, urgency, conflict | Villain reveals; action climaxes |
| Low-contrast (similar values) | Calm, depression, monotony | Riley's gray apartment in *Inside Out* |

## Saturation as Emotional Volume

```
HIGH SATURATION ─────────────────── LOW SATURATION
Energy · Youth · Fantasy · Joy      Realism · Age · Grief · Fatigue
```

---

## Color Temperature

| Color Temperature | Depth Effect | Emotional Association |
|------------------|-------------|----------------------|
| Warm (red, orange, yellow) | Advances (comes forward) | Energy, passion, warmth |
| Cool (blue, purple, green) | Recedes (moves back) | Calm, isolation, cold |

**Application:** Characters warmer/more saturated than backgrounds → reads forward automatically.

---

## Composition Tools

| Tool | Function |
|------|---------|
| Rule of Thirds | Place subjects at grid intersections (not dead center) |
| Leading lines | Environmental lines guide eye to focal point |
| Figure-ground contrast | High value contrast between subject and background |
| Negative space | Empty space used actively in the composition |
| Color temperature for depth | Warm foreground, cool background |

---

## Pixar Color Script Process

1. Establish emotional arc in color (beginning → end)
2. Set color keys for each major scene
3. Confirm character colors read in all environments
4. Establish contrast relationships (character vs background)

---

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| Hue is most important for readability | Value is most important; similar values can't be distinguished |
| Saturation always = quality | Low saturation communicates grief/realism; neither is better |
| Warm = advancing is wrong | Correct — warm colors advance; cool colors recede |
| Monochromatic = boring | Monochromatic = unity and restriction; powerful when intentional |
| Chromatic aberration = mistake | In Spider-Verse, it is a deliberate stylistic storytelling tool |
| Rule of thirds = mandatory | It is a guide that can be subverted intentionally (Miyazaki centering) |
