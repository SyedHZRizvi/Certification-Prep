# Module 4 Cheat Sheet: Storyboarding & Animatics 📋

---

## Shot Types (Board Reference)

| Shot | What It Shows | Primary Use |
|------|--------------|-------------|
| EWS | Character tiny in environment | Establish geography |
| WS | Full character | Action; context |
| MS | Waist up | Dialogue; interaction |
| CU | Face | Emotion; reaction |
| ECU | Eyes/detail | Maximum intensity |
| Insert | Specific object | Emphasis |
| POV | What character sees | Perspective |
| OTS | Over character's shoulder | Dialogue; spatial relationship |

---

## Transitions Quick Reference

| Transition | When to Use |
|-----------|------------|
| **Cut** | Always first choice; continuous action |
| **Dissolve** | Time passage; memory; dream |
| **Fade to Black** | Act break; significant time jump |
| **Wipe** | Comic effect; stylized (Star Wars) |
| **Match Cut** | Thematic parallel; elegant connection |
| **J-Cut** | Audio from next scene begins before visual |
| **L-Cut** | Audio from current scene continues over next |

---

## Camera Move Arrows

| Arrow Direction | Camera Move |
|---------------|------------|
| → or ← | Pan (horizontal) |
| ↑ or ↓ | Tilt (vertical) |
| → Inward | Push in (zoom effect toward subject) |
| ← Outward | Pull out (zoom away from subject) |

---

## Storyboard Panel Elements

1. The drawing (staging, characters, action)
2. Action description (what happens)
3. Dialogue (what is said)
4. Camera direction (arrows)
5. Shot number
6. Scene/sequence number

---

## The Animatic's Job

An animatic proves or disproves:
- [ ] Pacing — does the scene breathe or drag?
- [ ] Emotional timing — does the audience have time to feel?
- [ ] Action clarity — can the audience follow?
- [ ] Comedy timing — does the gag land?
- [ ] Sound/music sync — does the visual match the audio?

---

## Thumbnailing: The 3-Rule

> Draw at least **3** different compositions for any significant story beat before choosing.
> Rule 1 = most obvious. Rule 3 = usually most interesting.

---

## Board Pitch Structure

1. **Context** — where we are, what just happened
2. **Walk the boards** — describe action, voice dialogue, show emotion
3. **Justify key choices** — explain significant staging decisions
4. **Invite notes** — specific questions to guide director feedback

---

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| Storyboards = final artwork | Boards are rough planning documents |
| More shots = better scene | Scene efficiency = right shots, not most shots |
| Animatics only useful for comedy | Animatics test pacing, emotion, clarity in all genres |
| J-cut = jump cut | J-cut = audio from incoming scene starts before visual |
| OTS = wide shot of both characters | OTS = Over-the-Shoulder; one character in foreground |
