# Module 2 Cheat Sheet: Animation History & Visual Language 📋

---

## Animation History Timeline (Exam Version)

| Year | Event | Why It Matters |
|------|-------|---------------|
| 1878 | Muybridge's horse photographs | First motion study; birth of reference-based animation |
| 1887 | *Animal Locomotion* published | 781 plates; still used in animation schools |
| 1914 | *Gertie the Dinosaur* (McCay) | First animated character with personality |
| 1928 | *Steamboat Willie* (Disney) | Synchronized sound in animation |
| 1937 | *Snow White* (Disney) | First full-length animated feature |
| 1940 | *Fantasia* (Disney) | Peak of Disney classical style |
| 1942 | *Bambi* (Disney) | 12 Principles demonstrated at highest level |
| 1950 | *Gerald McBoing-Boing* (UPA) | Modernist limited animation; post-Disney aesthetic |
| 1960s | Hanna-Barbera TV era | Limited animation industrialized |
| 1986 | *Luxo Jr.* (Pixar) | First Pixar short; CGI personality proven |
| 1989–1999 | Disney Renaissance | *Little Mermaid, Lion King, Mulan, Tarzan* |
| 1995 | *Toy Story* (Pixar) | First feature-length CGI film |
| 2001 | *Spirited Away* (Ghibli) | First non-English animated film to win Oscar |
| 2018 | *Spider-Verse* (Sony) | Revolutionary hybrid visual style |
| 2021 | *Arcane* (Riot/Fortiche) | First animated series to win Primetime Emmy for animation |

## Disney vs Fleischer: Fast Comparison

| | Disney | Fleischer |
|-|--------|-----------|
| Style | Naturalistic, weighted | Rubber-hose, boneless |
| Rhythm | Story-driven | Jazz-synchronized |
| Emotion | Drama, sentiment | Vaudeville, adult |
| Modern heir | Pixar, DreamWorks | Spider-Verse, Adventure Time |

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| *Steamboat Willie* was first animated film | First with synchronized sound; McCay made animation earlier |
| *Toy Story* was first Pixar film | *Luxo Jr.* (1986) was Pixar's first short |
| *Spirited Away* won Best Picture | It won Best Animated Feature; first non-English film to do so |
| 180-degree rule cannot be broken | It can be broken *deliberately* for storytelling (Spider-Verse) |
| Limited animation = low quality | Limited animation is a different aesthetic, not a lesser one |
| "Mickey-Mousing" = Mickey Mouse design | It means synchronizing every musical beat to a visual action |
