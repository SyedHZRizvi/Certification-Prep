## Bouncing Ball: Principles It Teaches

| Principle | How the Ball Demonstrates It |
|-----------|------------------------------|
| Squash & Stretch | Widens on impact, elongates in flight (volume constant) |
| Slow-In/Slow-Out | Slow near arc top, fast at bottom |
| Arcs | Path is parabolic, never V-shaped or angular |
| Timing | Frame count = weight and material |
| Follow-Through | Successive bounces, smaller each time |
| Weight | More frames per bounce = heavier material |

## Bouncing Ball Timing by Material

| Material | Frames/Bounce | Squash Amount | Recovery |
|----------|--------------|---------------|---------|
| Ping-pong | 4–6 | Small | Very elastic |
| Rubber | 8–12 | Medium | Elastic |
| Basketball | 10–16 | Medium | Moderate |
| Golf ball | 6–10 | Minimal | Hard impact |
| Bowling ball | 20–32 | Minimal | Very little |
| Water balloon | 18–24 | Extreme | Breaks |
| Soap bubble | 16–24 | Almost none | Pops |

## Walk Cycle Key Positions

```
CONTACT: Foot lands; body LOWEST; opposite arm forward
DOWN: Body drops as weight transfers
PASSING: Free leg swings past planted leg; body HIGHEST
UP: Standing leg pushes off; body rises
```

**Arm swing:** Opposite arm from leading leg = always forward

## Walk Cycle Timing

| Type | Frames/Step | Character |
|------|------------|-----------|
| Snappy | 8–10 | Young, energetic, excited |
| Normal | 12 | Standard adult |
| Tired | 16–20 | Exhausted, sad, heavy |
| Menacing | 20–24 | Threatening, deliberate |

## First Short Production Checklist

**Pre-production:**
- [ ] Story = one clear sentence
- [ ] Storyboards complete
- [ ] Animatic with audio complete

**Production:**
- [ ] Reference recorded and studied
- [ ] Key poses approved before in-betweening
- [ ] Timing matches emotional intent

**Post-production:**
- [ ] Correct render codec (not H.264 for masters)
- [ ] Audio mixed and synced
- [ ] Files backed up (3-2-1 rule)

## Self-Critique Framework (5 Steps)

1. **Watch without sound** — does it read from movement alone?
2. **Cover character, watch background** — is staging clear from negative space?
3. **Play at half speed** — do in-betweens hold up? Any pops or slides?
4. **Show to fresh eyes** — what story do they see?
5. **Wait 48 hours, watch again** — fresh eyes reveal what familiarity hides

## Exam Traps

| Common Error | Correct Fact |
|-------------|-------------|
| Ball is fastest at arc top | Ball is SLOWEST at arc top (almost stationary); fastest at bottom |
| Arms and legs swing same side | Opposite arm from leading leg swings forward |
| Body is lowest at passing position | Body is HIGHEST at passing; lowest at contact/down |
| Volume changes with squash | Volume stays CONSTANT — wider when squashed, narrower when stretched |
| V-shaped bounce paths are OK | Arcs must be parabolic curves, never V-shapes |
