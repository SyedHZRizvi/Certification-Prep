# Module 1 Cheat Sheet: Disney's 12 Principles 📋

> Print this. Put it next to your drawing table or monitor. Know it cold.

---

## The 12 Principles at a Glance

| # | Principle | One-Line Definition | Key Constraint |
|---|-----------|--------------------|----|
| 1 | **Squash & Stretch** | Deform under force | Volume stays constant |
| 2 | **Anticipation** | Prepare for the action | Earns the payoff |
| 3 | **Staging** | One clear idea at a time | Silhouette must read |
| 4 | **Straight-Ahead vs Pose-to-Pose** | Two production methods | Use both; each has strengths |
| 5 | **Follow-Through & Overlapping** | Parts continue after main stops | Inertia of mass |
| 6 | **Slow-In / Slow-Out** | Natural objects ease in/ease out | S-curve in graph editor |
| 7 | **Arcs** | Living things move in curves | Joints rotate → arcs |
| 8 | **Secondary Action** | Supporting action enriches primary | Never compete with primary |
| 9 | **Timing** | Frame count = weight/emotion | More frames = heavier/slower |
| 10 | **Exaggeration** | Push beyond literal reality | Serve emotional truth |
| 11 | **Solid Drawing** | 3D form in 2D medium | Weight, balance, overlap |
| 12 | **Appeal** | Engaging quality, not cute | Clarity of character |

## Timing Quick Reference

| Frame Count | Impression |
|------------|------------|
| 2–4 frames | Shock, impact, comedy snap |
| 6–8 frames | Quick energetic action |
| 12–16 frames | Normal movement |
| 24–32 frames | Deliberate, controlled |
| 48–72 frames | Heavy, tired, solemn |
| 96+ frames | Grief, ceremony, dreamlike |

---

## Production Method Comparison

| | Straight-Ahead | Pose-to-Pose |
|-|---------------|-------------|
| **How** | Frame 1 → 2 → 3 in order | Key poses first → fill in-betweens |
| **Strength** | Organic, spontaneous energy | Controlled timing, dialogue sync |
| **Best for** | Effects, creature motion, comedy | Character performance, dialogue |
| **Risk** | Volume/proportion drift | Mechanical in-betweens |
| **Most used in** | Traditional 2D (Disney hand) | Digital 3D (Pixar norm) |

## Key Case Studies: Which Principle to Spot

| Film Scene | Primary Principle | Why |
|-----------|-------------------|-----|
| Bambi, deer on ice | Slow-In/Slow-Out | Ice removes friction; extreme ease reveals physics |
| Bambi, deer running | Overlapping Action | Ears lag behind head direction changes |
| Into the Spider-Verse, Miles's leap | Anticipation | Long pause before the leap earns the payoff |
| Spider-Verse, different fps per character | Timing | 12fps vs 24fps communicates character difference |
| Spirited Away, Chihiro not crying | Secondary Action | Trembling lip supports (not competes with) stillness |
| The Incredibles, Mr. Incredible lifts | Squash & Stretch | Volume-conserving deformation communicates weight |
| Spirited Away, Chihiro small in frame | Staging | Character size = emotional state (overwhelmed) |
| Lion King, Scar | Appeal | Clarity of villainous character; not cute, but engaging |

## Exam Traps, Know These

| Common Mistake | Correct Fact |
|---------------|-------------|
| Squash & Stretch = exaggeration | They are related but distinct; S&S is specifically volume deformation |
| Straight-Ahead is always better | Pose-to-Pose is better for dialogue; use both |
| Appeal = cuteness | Appeal = engaging clarity of character; villains have appeal |
| Secondary Action replaces primary | Secondary supports without competing |
| Arcs are optional in 3D software | 3D software defaults to straight paths; arcs must be manually enforced |
| Slow-out = slow animation | Slow-out is deceleration at the END of a move, not slowness throughout |
| 12 Principles are Disney-specific | They apply universally: 2D, 3D, stop-motion, game animation |

---

## The "Bambi Ears Test"

> If a character's secondary elements (hair, coat, ears, tail) stop moving at exactly the same moment as the primary body, they have failed the Bambi Ears Test. Secondary elements must **lag, overlap, and follow-through** relative to the primary body.

## Staging Checklist

- [ ] Silhouette reads as a clear black shape on white
- [ ] One center of attention per frame
- [ ] Highest contrast on the most important element
- [ ] Character size communicates emotional state
- [ ] Camera angle reinforces power dynamic (low = powerful, high = vulnerable)
