## Detailed answer rationales

For each question, the correct answer is stated with the framework/citation it tests; key incorrect options are explained as the *trap* they represent. Use this for targeted weakness-finding after scoring.

**Q1. Answer: A — MTS.** Standardized + forecast-driven + ships from stock = textbook Make-to-Stock. Module 1, Hill (1985) decoupling-point taxonomy. Wrong: B/C/D require an *order trigger* somewhere in the chain; standardized garden fittings to forecast don't.

**Q2. Answer: C — Industry-leading battery life.** Order winner = differentiator that wins the sale; if every phone has a working camera, Bluetooth, and SIM slot, those are *qualifiers*. Hill (1985); Module 1.

**Q3. Answer: D — Sub-assembly / module level.** ATO's decoupling point is at the module level — modules forecast-built, final assembly order-triggered. Module 1. Wrong A: FG would make it MTS. Wrong B: design = ETO. Wrong C: raw = MTO.

**Q4. Answer: A — Delaying final product differentiation until customer order received.** Module 1, HP printer postponement case (Lee, Billington & Carter, HBR 1994). Wrong B: that's a *delay* — different word, different concept. Wrong C: safety stock, unrelated. Wrong D: outsourcing assembly is a make/buy decision.

**Q5. Answer: D — Operate ON the diagonal.** Hayes & Wheelwright (1984): align process to volume-variety. Wrong A is the off-diagonal trap; the matrix says off-diagonal is structurally wrong (with deliberate exceptions like Zara).

**Q6. Answer: A — People, Planet, Profit.** Elkington (1994 *California Management Review*; book 1997). The other options are scrambled jargon.

**Q7. Answer: A — Delphi method.** Delphi is qualitative (anonymous expert rounds). The other three are quantitative time-series or causal methods.

**Q8. Answer: D — Trend, Cyclical, Seasonal, Random.** TCSR — the 4 components of demand (Module 2). The other options scramble unrelated supply-chain concepts.

**Q9. Answer: C — Over multiple years.** Cyclical = multi-year (business cycle, construction cycle). Seasonal = within-year. Module 2, ASCM Dictionary entry.

**Q10. Answer: B — 75.** 4-month SMA = (60+70+80+90)/4 = 300/4 = 75. Wrong D (80): that's just the latest data point, not the average.

**Q11. Answer: D — 126.** WMA = 0.2·100 + 0.3·120 + 0.5·140 = 20 + 36 + 70 = 126. Wrong A (118): order of weights reversed. Wrong B (122): arithmetic error.

**Q12. Answer: D — 540.** F_next = 0.4·600 + 0.6·500 = 240 + 300 = 540. Wrong A (520): arithmetic. Wrong C (560): used α on prior forecast not demand.

**Q13. Answer: C — Increase α.** Higher α = more responsive to recent demand. Module 2; Hyndman & Athanasopoulos (2021). Wrong A: decreasing α makes it less responsive.

**Q14. Answer: D — 8.2.** Σ|e| = 8+12+10+5+6 = 41; MAD = 41/5 = 8.2. Wrong A (5): probably min(|e|). Wrong B (12): max(|e|).

**Q15. Answer: C — +5.** Σe = -8+12-10+5+6 = 5. Bias is the *signed* sum. Wrong A (-5): sign error. Wrong B (+25): used absolute values.

**Q16. Answer: C — 4.** Standard CPIM/Wallace cutoff is |TS| > 4 (some texts use ±6 — both accepted). Module 2.

**Q17. Answer: A — Percentage.** MAPE = mean absolute *percent* error. Wrong B (squared): that's MSE. Wrong C (absolute units): that's MAD.

**Q18. Answer: D — Demand variability amplification upstream.** Bullwhip definition (Lee, Padmanabhan & Whang, 1997). Wrong A, B, C are unrelated.

**Q19. Answer: A — Share forecasts between trading partners.** CPFR (VICS, 1998). Wrong B/C/D misrepresent CPFR's purpose.

**Q20. Answer: B — Delphi / market research.** No history = qualitative methods only. Time-series and regression need data. Module 2.

**Q21. Answer: C — Product / Portfolio Review.** Wallace 5-step canonical Step 1. Module 3.

**Q22. Answer: B — Supply Review.** Operations leads Step 3. Sales/Marketing leads Step 2 (Demand). CEO leads Step 5.

**Q23. Answer: A — Executive S&OP.** Step 5 — the CEO chairs the executive review. Module 3.

**Q24. Answer: B — Monthly family.** S&OP cadence is monthly; aggregation is family-level. SKU planning belongs to MPS. Module 3.

**Q25. Answer: D — Mature, financially integrated form of S&OP.** IBP (Oliver Wight) extends S&OP with financial and strategic integration on 24-36 month horizon. Wrong A: IBP doesn't *replace* S&OP — it's the matured form.

**Q26. Answer: C — Level.** Level = constant production rate; inventory absorbs demand variation. Module 3.

**Q27. Answer: B — Chase.** Perishables = no storage; production must chase demand exactly. Module 3.

**Q28. Answer: B — Step 2.** The Demand Review (Step 2) produces the *unconstrained* demand plan. Constraints come from Supply Review (Step 3).

**Q29. Answer: D — MPS.** Below S&OP in the planning hierarchy sits MPS. Then MRP. Then PAC.

**Q30. Answer: A — Highest (long-range, strategic).** Resource Planning sits at the top of the capacity hierarchy. RCCP, CRP, I/O Control descend below it.

**Q31. Answer: B — Finished goods (end items).** MPS plans the anchor level. MRP plans components.

**Q32. Answer: D — Dependent.** MRP plans dependent (BOM-calculated) demand. MPS plans independent demand. Module 4.

**Q33. Answer: A — DTF is closer.** Demand Time Fence is the *near-term* frozen zone. PTF is further out. Memorize: DTF < PTF in time.

**Q34. Answer: D — Firm customer orders only.** Inside the DTF, the forecast is *ignored* — only firm orders drive PAB. Highest-frequency exam trap. Module 4.

**Q35. Answer: B — 150.** ATP = MPS receipt − orders against it = 400 − 250 = 150. Module 4.

**Q36. Answer: B — 1,200.** GR = MPS qty × BOM qty = 600 × 2 = 1,200 wheels. Module 4.

**Q37. Answer: C — 300.** Net req = Gross − On-hand − Sch receipts = 800 − 200 − 300 = 300. Module 4.

**Q38. Answer: C — Week 4.** Planned receipt week 6 − LT 2 weeks = release week 4. Module 4.

**Q39. Answer: A — Ensure each item is processed once, with all parent demand netted.** Module 4. Standard MRP optimization.

**Q40. Answer: B — Push an open order later.** Reschedule out = move out in time. Reschedule in = pull earlier. Module 4.

**Q41. Answer: A — The exact net requirement each period.** L4L. Wrong B: that's FOQ. Wrong C: that's EOQ. Wrong D: forecast ≠ net req.

**Q42. Answer: B — ATO.** Two-level MPS: modules at one level + final assembly at another. Standard ATO architecture. Module 4.

**Q43. Answer: B — Considers replanning of capacity and materials.** CTP = "Capable-to-Promise" — considers what we *could* deliver if we replan. ATP = current schedule only.

**Q44. Answer: A — Random.** TCSR = Trend, Cyclical, Seasonal, Random. Module 2.

**Q45. Answer: D — Qualitative method.** Sales force composite is qualitative. The others are quantitative.

**Q46. Answer: C — The minimum criteria a product must meet to be considered.** Order qualifier (Hill, 1985). Order winner = differentiator.

**Q47. Answer: A — Hill's principle (winner-to-qualifier shift).** Today's winner becomes tomorrow's qualifier as competitors catch up. Module 1.

**Q48. Answer: B — Unconstrained at Step 2, then reconciled.** Step 2 demand plan must be unconstrained; Operations cannot cap demand at this stage. Reconciliation happens in Step 4 (Pre-S&OP) and Step 5 (Executive). Wrong A: constrains too early.

**Q49. Answer: C — Daily shift schedule.** S&OP outputs are at the family level for 12–24 months. Daily shift schedules are PAC outputs.

**Q50. Answer: C — Plan is replanned every cycle.** Rolling horizon — the plan is never frozen for the full year; each cycle revises forward.

**Q51. Answer: B — Uses level base + overtime/contract for peaks.** Hybrid combines stability (level core) with flex (overtime overlay). Module 3.

**Q52. Answer: C — MAD is in units; MAPE is a percent.** Module 2 — most-tested forecast-metric distinction.

**Q53. Answer: C — FOQ or POQ.** Large supplier minimums → fixed-order quantity or period-order quantity, not L4L (which would order under the minimum).

**Q54. Answer: C — Very responsive but noisy.** High α = highly responsive; the cost is noise sensitivity. Module 2.

**Q55. Answer: C — The link between a component requirement and its parent demand.** Pegging traceability. Module 4.

**Q56. Answer: D — More accurate at higher aggregation and longer time buckets.** The forecast aggregation principle. Module 2.

**Q57. Answer: A — Lot-sizing rules.** Lot-sizing rules govern how planned orders are grouped per period. Wrong B: time fences govern *when* changes can be made.

**Q58. Answer: B — The frozen near-term period where commitments are made.** DTF rationale. Material/capacity is locked, so change is expensive.

**Q59. Answer: A — Investigate — the model is biased.** |TS| > 4 (or ±6) is the alarm threshold. Module 2.

**Q60. Answer: D — S&OP → MPS → MRP → PAC.** Standard top-down planning hierarchy.

**Q61. Answer: D — A structural mismatch between volume/variety and process.** Off-diagonal on Hayes-Wheelwright. Module 1.

**Q62. Answer: C — Bringing offshored production back to the domestic market.** Reshoring definition. 2020–2026 trend driven by tariffs, geopolitical risk, automation cost-shift.

**Q63. Answer: A — Bias.** Cumulative forecast error = Σ e_t = bias. Module 2.

**Q64. Answer: B — Between 0 and 1.** α ∈ (0, 1). Wrong A (>1): mathematically invalid. Wrong D (=1): valid but means "use only the most recent demand" — extreme case.

**Q65. Answer: B — Product → Demand → Supply → Pre-S&OP → Executive.** Wallace 5-step. Module 3.

**Q66. Answer: A — Postponement.** HP printer chassis + country-specific kit at regional DC. Module 1, classic case.

**Q67. Answer: B — High volume, low variety.** Continuous flow's diagonal sweet spot. Module 1.

**Q68. Answer: A — ATO architecture with postponement.** Mass customization implementation pattern. Module 1.

**Q69. Answer: D — Moved or modified automatically by the system.** Firm planned order is locked from auto-replanning. Module 4.

**Q70. Answer: B — The sum of all lead times on the longest material path through the BOM.** Cumulative LT = critical-path LT through the BOM. Module 4.

**Q71. Answer: C — Reduces nervousness but limits responsiveness to demand changes.** PTF trade-off. Module 4.

**Q72. Answer: C — Capacity, material, and supplier commitments.** S&OP supply plan = how Operations will deliver. Module 3.

**Q73. Answer: A — Capacity limits.** Unconstrained = no capacity ceiling at Step 2. Module 3.

**Q74. Answer: D — 3.** Low-level code = the *deepest* level where an item appears. The tube set appears at level 3 in Bike B, so its low-level code is 3.

**Q75. Answer: D — Independent demand.** Demand from outside the plant = independent. Module 4.
