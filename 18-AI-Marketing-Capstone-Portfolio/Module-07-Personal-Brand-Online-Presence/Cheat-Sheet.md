# 📋 Module 7 Deliverable Rubric: Personal Brand

> Print this. Grade before considering this module done.

---

## 🎯 The Deliverables

```
□ LinkedIn rebuilt: headline, About, Featured (4–6 pins), banner
□ Portfolio site LIVE on a public URL with all 5 capstones
□ Content distribution playbook chosen (channels + cadence)
□ First 10+ LinkedIn posts drafted
□ First 5 LinkedIn posts published
□ 60-second cold-open pitch written + practiced
□ 5 network reachouts sent
□ Custom domain (optional but recommended)
```

---

## 📐 Section-By-Section Rubric

| Component | Failing | Passing | Excellent |
|-----------|---------|---------|-----------|
| **LinkedIn Headline** | Job title default | Custom but generic | Pattern A/B/C with CTA (Call to Action) |
| **About Section** | Corporate-speak | Custom, listy | Story open + 5 capstones + CTA |
| **Featured Pins** | 0–2 pins | 3 pins | 4–6 pins with custom titles |
| **Banner** | Default blue | Custom basic | Custom on-brand |
| **Portfolio Site** | Not live | Live, basic | Live + all 5 capstones + custom domain |
| **Case Studies** | Missing or inconsistent | 3+ present | All 5 in P-C-A-R-L |
| **Post Drafts** | 0–4 drafts | 5–9 drafts | 10+ drafts |
| **Live Posts** | 0–2 live | 3–4 live | 5+ live with engagement |
| **Cadence** | Random | Weekly | 3–5x/wk fixed days |
| **Cold-Open Pitch** | None | Drafted | Written + practiced 3+ times |
| **Network Reach** | None | 1–4 reachouts | 5+ messages sent |

### To Pass: ≥8 cells Passing+
### Recruiter-Grade: ≥9 Excellent

## 🏆 LinkedIn Headline, 3 Patterns

```
PATTERN A, Outcome + Audience:
  "Help [audience] do [outcome] with [unique angle]"
  
PATTERN B, Capstone-Anchored:
  "[Role] | Built [capstone], [where to find]"

PATTERN C, Vibe + Skill:
  "[Identity] learning to [skill] • portfolio in Featured"
```

Pick B if entry-level. A if you have a clear niche. C if indie/creator track.

---

## 📝 About Section Template

```
[Opening: 1 paragraph, 80 words]
[A specific moment from your most interesting capstone]

[Middle: 1-2 paragraphs, 150 words]
[What you do + 5 capstones listed with brief value]

[Closing: 1 paragraph, 70 words]
[Where to find work + how to reach you]
```

---

## 📌 Featured Pins (4–6)

```
1. Portfolio site link
2. Most impressive capstone (paid campaign usually)
3. Most technical capstone (AI agent or attribution)
4. LinkedIn carousel about a capstone
5. (optional) YouTube/Loom channel
6. (optional) Build-in-public post
```

---

## 📅 Posting Cadence

```
Mon, Long-form LinkedIn (story-driven case)
Tue, Twitter thread (3-tweet recap)
Wed, Comment-mining day (engage 10 posts)
Thu, Short LinkedIn (1 lesson, 80 words)
Fri, Carousel OR YouTube/Loom
Weekend, Off / build-in-public snippets
```

---

## 📝 The 30-Post Distribution

```
5  Capstone deep-dives (1 per capstone)
8  Lessons learned ("I learned X the hard way")
6  Behind-the-scenes ("Here's the process")
4  Contrarian takes
3  Resource shares
2  Question / polls
2  Milestone posts
```

---

## 🚨 Anti-Patterns To Avoid

- ❌ Rebuilt LinkedIn but never posted
- ❌ "I'll Webflow forever" trap
- ❌ Engagement-baiting ("Comment YES if you agree!")
- ❌ Generic posts (no numbers, no screenshots)
- ❌ Inconsistent voice across capstones
- ❌ No portfolio link in headline
- ❌ Hiding the "weaker" capstones
- ❌ Polishing forever before shipping

## 🎤 60-Second Cold-Open Pitch

```
"I'm a marketer focused on [niche]. Over the last 6 weeks I shipped
5 portfolio pieces, [list one-line]. I'm currently [status]. Most
interested in roles that touch [3 things]. Happy to walk you through
any of the case studies."
```

Memorize. Use 30+ times.

---

## 🏆 Power Phrases In Posts

When opening:

- ✅ "Last [time period], I [specific action]..."
- ✅ "3 hours into [project], I hit [problem]..."
- ✅ "I used to think [X]. Now I think [Y]. Here's why:"

When CTA'ing:

- ✅ "Full write-up + screenshots: [link]"
- ✅ "If you're working on [similar problem], DM me"
- ✅ "Repo + Loom in Featured ↓"

Avoid:

- ❌ "Comment YES if you agree!"
- ❌ "Tag a friend who needs this!"
- ❌ "Like and follow for more!"
