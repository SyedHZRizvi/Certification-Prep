## 🎯 Model Answers + Grading Guidance

### Q1: Excellent looks like: **A**
A scenario you've built that fires when triggered. The whole capstone is "I built this thing." If you didn't build it, the capstone doesn't exist.

### Q2: Excellent looks like: **A**
5+ modules + at least one Router. Below that, the workflow is too simple to demonstrate systems thinking. Above 12 modules, you may be overcomplicating.

### Q3: Excellent looks like: **A**
Router with 3+ branches. This is the move that shows you're thinking about IF/THEN logic, not just linear pipes.

### Q4: Excellent looks like: **A**
Enrichment running + producing real data. Senior marketers in 2026 don't ship lead routing without enrichment, it's the data layer that makes scoring useful.

### Q5: Excellent looks like: **A**
Scoring formula with 3+ attributes, documented weights. *"30 points for company size 51+, 30 for SaaS/Fintech/HealthTech, 40 for high-intent keywords in message."* Reviewable, defensible.

### Q6: Excellent looks like: **A**
HOT/WARM/COLD paths all tested with different inputs. Demonstrates rigor.

### Q7: Excellent looks like: **A**
1-page architecture diagram. Cover image for the case study. Hand-drawn (Excalidraw) is FINE even preferred over polished because it signals you can think visually.

### Q8: Excellent looks like: **A**
Loom uploaded + publicly linked. Without it, the capstone is text-only, which is exactly the wrong format for an "I built a thing" piece.

### Q9: Excellent looks like: **A**
Hook in the first 30 seconds. *"Here's a 7-step automation that takes a lead from form to Salesforce + Slack + Klaviyo in under 30 seconds, let me show you."*

### Q10: Excellent looks like: **A**
Public JSON link. This is the "github repo" equivalent. Without it, the recruiter can't verify the work exists.

### Q11: Excellent looks like: **A**
Sanitized. Verified. The most common embarrassing portfolio failure is exporting with API keys included. Check before sharing.

### Q12: Excellent looks like: **A**
Full P-C-A-R-L. Same structure as every other capstone. Consistency = trust.

### Q13: Excellent looks like: **A**
Config-level screenshots. Not just the pretty canvas view, show the inside of each module so the reader sees what fields you mapped where.

### Q14: Excellent looks like: **A**
Quantified value. *"Replaces ~6 manual hours/week, valued at ~$X based on senior SDR rate."* Mapping work to money = senior thinking.

### Q15: Excellent looks like: **A**
Actually fires. The Loom proves this. Don't ship a scenario that doesn't actually run.

### Q16: Excellent looks like: **A**
Error handler / retry / alert path. Junior marketers ship happy-path-only. Senior ones ship with error handling. This is a strong signal.

### Q17: Excellent looks like: **A**
Public URL. Linkable from LinkedIn. Indexable in Google.

### Q18: Excellent looks like: **A**
Posted + engaged. Marketing-automation content does well on LinkedIn, niche-y enough to be interesting, technical enough to filter out tourists.

### Q19: Excellent looks like: **A**
Every architecture decision documented. *"Used Router instead of multiple Filters because branching keeps shared steps in one place."*

### Q20: Excellent looks like: **A**
Real or realistic use case. Toy "send Slack when someone fills the form" examples don't sell. Tie it to a business problem.

### Q21: Excellent looks like: **A**
3+ v2 ideas. *"v2 would add LLM-based message classification, automatic CRM dedup, and a feedback loop where rejected leads adjust the scoring."*

### Q22: Excellent looks like: **A**
Free tier or cost documented. Recruiters care about cost-consciousness.

### Q23: Excellent looks like: **A**
Multiple edge cases tested. *"What if email is missing? What if Hunter rate-limits? What if same form submitted twice within 30 sec?"*

### Q24: Excellent looks like: **A**
Diagram matches implementation. Drift between diagram and reality = unprofessional.

### Q25: Excellent looks like: **A**
6–10 hours. The capstone is calibrated to this range.

### Q26: Excellent looks like: **A**
Portable JSON. Someone else can import + customize. This is what makes the export valuable.

### Q27: Excellent looks like: **A**
Live execution shown in the Loom. The "magic moment" where you submit the test form and watch the data fan out across Slack / Notion / etc.

### Q28: Excellent looks like: **A**
Practiced 3+ times. Same drill as every capstone.

- 16–21 A's → ✅ Solid. Polish C/D items.
