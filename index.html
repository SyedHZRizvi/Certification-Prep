02 · Pick your goal

Two world-class certifications.
One thoughtfully-designed curriculum.

Start with whichever fits your timeline. Doing both? We recommend Scrum Master first — it's faster, cheaper, and builds the foundation.

Recommended start 🏃

Scrum Master

Professional Scrum Master I (PSM I)

⏱ 4 weeks 💵 $200 exam 🎯 85% to pass

Master the Scrum framework, ace the official Scrum.org exam, and earn a lifetime credential with no renewal fees.

  • ✓ The Agile Manifesto + Empiricism
  • ✓ 3 Roles · 5 Events · 3 Artifacts
  • ✓ Servant leadership patterns
  • ✓ Scaling (Nexus, LeSS, SAFe)
  • ✓ 3 mock exams + 80-card flashcard deck
Start the Scrum track →
Most respected PM cert 📊

PMP

Project Management Professional

⏱ 12 weeks 💵 $405-555 exam 🎯 3 domains

Pass the world's most-trusted project management exam with the PMP-2021 mindset baked into every lesson.

  • ✓ People domain (42%) — the biggest piece
  • ✓ Process domain (50%) — scope, schedule, cost
  • ✓ Earned Value formulas drilled to muscle memory
  • ✓ Risk · Quality · Procurement · Stakeholders
  • ✓ 3 mock exams + 70-card flashcard deck
Start the PMP track →
🧭

Suggested order if you're doing both

Scrum Master first (weeks 1-4), then PMP (weeks 5-16). The Scrum content covers ~50% of PMP's agile/hybrid section, so you'll be ahead of the game. Already certified in Scrum? Skip straight to PMP.
