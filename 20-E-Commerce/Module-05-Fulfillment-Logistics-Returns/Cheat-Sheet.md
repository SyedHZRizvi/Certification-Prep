# 📋 Module 5 Cheat Sheet: Fulfillment, Logistics & Returns

> Print and tape. Fulfillment economics decide whether ads pay back.

---

## 🏗️ The Fulfillment Stack (MEMORIZE)

```
ORDER  →  OMS  →  WMS  →  TMS  →  CARRIER  →  LAST-MILE  →  CUSTOMER
                                                              ↓
                                                          RETURNS
                                                          REVERSE LOGISTICS
```

```
OMS  = Routing brain (where to ship from)
WMS  = Warehouse ops (pick/pack)
TMS  = Carrier selection (whom to ship with)
```

## 🆚 FBA vs FBM Cheat Sheet

| Dimension | FBA | FBM |
|-----------|-----|-----|
| Fulfiller | Amazon | You/your 3PL |
| Prime badge | Yes | Limited (SFP restricted) |
| Buy Box win | Higher | Lower |
| Fees | ~15% + storage | Lower commission |
| Inventory control | Limited | Full |
| Best for | Standard, fast-moving | Bulky, slow, branded |

---

## 🎁 Buy with Prime + MCF (2023-2024)

```
Buy with Prime → DTC site uses FBA inventory + Prime badge
MCF            → FBA inventory fulfills any marketplace (Walmart, eBay, Etsy, TikTok Shop)
```

Buy with Prime fees:

- Fulfillment: FBA-equivalent ($3-$15/unit)
- Storage: standard FBA
- Payment processing: +3%
- Returns: $1-$3

🚨 When to skip: luxury, unboxing-moat, full data ownership.

---

## 💰 FBA Fee Structure (2026)

```
Fulfillment fee     → $3-$15/unit (size/weight)
Storage (Jan-Sept)  → $0.87/cubic foot
Storage (Oct-Dec)   → $2.40/cubic foot (peak surcharge!)
Long-term storage   → > 365 days adds fees
Returns processing  → $1-$3
```

---

## 📦 Free Shipping Threshold Math

```
Threshold = (Avg Shipping Cost / CM %) × 1.4

Example: $9 shipping, 50% CM
$9 / 0.50 × 1.4 = $25 minimum
→ Round to $50 marketing threshold
```

🧠 **49% of cart abandonments** cite shipping cost (Baymard 2024).

| Carrier | Sweet spot |
|---------|-----------|

| FedEx | Express, B2B |

## 🔄 Returns Stack

| Vendor | Best for |
|--------|----------|
| Loop Returns | DTC Shopify Plus (most-adopted) |
| Happy Returns (PayPal) | UPS Store drop-off network |
| AfterShip Returns | Multi-channel multi-region |
| Returnly (Affirm) | DTC payment-tied refunds |
| Narvar | Enterprise multi-carrier |

🧠 **Easy returns lift LTV 20-50%** (Loop 2024).

Return rates by category:

- Apparel: **30-40%** (highest)
- General DTC: 18-25%
- Furniture: 5-10% (but expensive when they happen)

---

## ♻️ Return Inspection Grades

```
A, Restock to inventory
B, Liquidate (Optoro, B-Stock @ ~30-50% of retail)
C, Refurbish
D, Dispose / recycle
```

## 🌐 Cross-Border Fulfillment

| Option | When |
|--------|------|

## 🇪🇺 EU Right of Withdrawal

```
Source: Consumer Rights Directive 2011/83/EU, Article 9
Window: 14 days no-questions-asked
Mandatory; member states implement
```

---

## ⚡ Failure Modes

- ❌ 3PL onboarding skipped due-diligence → 14% mis-ship in peak
- ❌ No OMS at 3+ channels → daily oversells
- ❌ FBA overstocked Q4 → $2.40 peak surcharge
- ❌ Apparel without returns budget → 35% return shock
- ❌ Free shipping promised, threshold too low → CM erosion
- ❌ Cross-border without IOSS → 30-50% conversion drop
- ❌ Single-carrier reliance → labor disruption = chaos
