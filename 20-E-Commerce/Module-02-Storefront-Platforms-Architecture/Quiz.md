## 🎯 Answers + Explanations

### Q1: **A. Microservices, API-first, Cloud-native, Headless**
The MACH Alliance definition. The four architectural principles for modern composable commerce.

### Q2: **B. $1M-$250M**
Shopify Plus's documented sweet spot. Below $1M, Shopify Basic suffices; above $250M, composable or Adobe Commerce often makes more sense.

### Q3: **C. 2018**
Adobe acquired Magento for $1.68B in 2018. The rebrand to "Adobe Commerce" came in 2021.

### Q4: **B. Shopify's template language**
Liquid was created by Tobi Lütke in 2006 and is now open-source. Adobe Commerce uses PHTML.

### Q5: **B. Shopify's React/Remix headless framework + Cloudflare-edge hosting**
Hydrogen is the framework (React + Remix, GA Jan 2023); Oxygen is the hosting (on Cloudflare's edge since 2022).

### Q6: **B. 2020**
The MACH Alliance was founded by commercetools, Contentstack, EPAM, and Valtech in 2020.

### Q7: **B. Different — headless decouples frontend from one engine; composable composes multiple best-of-breed services**
Headless ≠ composable. You can be headless (Shopify Hydrogen) without being composable (still uses Shopify's monolithic engine).

### Q8: **B. Demandware**
Founded 1998, IPO 2012, acquired by Salesforce 2016 for $2.8B. Rebranded as Salesforce B2C Commerce Cloud.

### Q9: **B. SCAPI is the newer RESTful API; OCAPI is the legacy**
SCAPI = Shopper Commerce API + Customer Commerce API (newer, RESTful). OCAPI = Open Commerce API (older). Salesforce B2C Commerce Architect cert tests this heavily.

### Q10: **B. Adobe Commerce (or composable)**
Complex B2B price-list logic + 5 international storefronts exceed what Shopify Plus does cleanly in 2026. Adobe Commerce B2B or composable is the call.

### Q11: **B. Merchants where the platform handles all cardholder data**
SAQ A = lowest burden, applies when card data never touches the merchant's systems (Shopify, BigCommerce, Salesforce).

### Q12: **B. March 2025**
PCI-DSS v4.0 was published in March 2022; mandatory compliance became March 2025.

### Q13: **C. GPL-3**
Magento Open Source is GPL-3 licensed. Adobe Commerce (the commercial version) has a different proprietary license.

### Q14: **B. "Sections everywhere" and JSON templates**
Online Store 2.0 (2021) introduced sections-everywhere model and JSON templates, replacing the old vintage themes architecture.

### Q15: **C. Strategic tech differentiation + best-of-breed across the stack**
Composable's value isn't lower cost — it's higher. The value is best-of-breed components + replace-any-component flexibility.

### Q16: **B. Composable / MACH-aligned commerce engine**
commercetools is the most-adopted MACH-Alliance composable commerce engine. Munich-based, founded 2006.

### Q17: **B. Stay on Shopify and migrate to Hydrogen + Markets + B2B**
Allbirds did NOT replatform. They evolved within Shopify — refactored theme onto Online Store 2.0, adopted Hydrogen for key pages, used Shopify Markets for international, stayed on Shopify B2B module.

### Q18: **A. PWA Studio**
PWA Studio (React + Apollo on Adobe Commerce's GraphQL API) is Adobe Commerce's headless framework.

### Q19: **B. $250K-$1.6M**
The documented 3-year TCO range for Shopify Plus at this GMV stage. Adobe Commerce equivalent is $950K-$4.95M.

### Q20: **A. Shopify's design system**
Polaris (polaris.shopify.com) covers components, content guidelines, accessibility, merchant-facing UX patterns.

### Q21: **C. Composable / MACH**
$300M GMV with B2B+B2C+omnichannel+tech differentiation needs is squarely in composable territory. Shopify Plus would work but composable enables the strategic differentiation question.

### Q22: **B. Unlimited frontend flexibility + performance via edge caching**
Headless lets you use any frontend framework and serve via edge networks, optimizing performance and design freedom.

### Q23: **B. 4-12%**
Shopify's 2024 commerce trends report puts Shop Pay's conversion lift at 4-12% above industry baseline checkout. Baymard 2024 ranks it as the fastest checkout on the internet.

### Q24: **B. Replatforming a $20M brand to enterprise platforms "because we'll scale" — paying $1-2M before reaching the scale**
The most expensive mistake. By the time scale arrives, requirements have changed and the platform may no longer fit.

### Q25: **C. SAQ D (full burden)**
Self-hosted Magento Open Source = you control cardholder data infrastructure = full PCI scope = SAQ D. Managed Adobe Commerce Cloud = SAQ A-EP.
